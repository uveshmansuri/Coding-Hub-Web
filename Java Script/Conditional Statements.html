Conditional Statements

Conditional statements allow you to execute specific blocks of code based on certain conditions. They are essential for controlling the flow of execution in your program.

The if Statement

The if statement evaluates a condition and, if the condition is true, executes the code block within it.

Syntax:


if (condition) {
    // code block to be executed if condition is true
}

Example:


let number = 10;
if (number > 5) {
    console.log("The number is greater than 5.");
}

In this example, since number is 10, which is greater than 5, the message will be printed.

The if-else Statement

The if-else statement allows you to specify an alternative block of code to execute if the condition is false.

Syntax:


if (condition) {
    // code block if condition is true
} else {
    // code block if condition is false
}

Example:


let number = 3;
if (number > 5) {
    console.log("The number is greater than 5.");
} else {
    console.log("The number is not greater than 5.");
}

Here, since number is 3, which is not greater than 5, the second block of code is executed, and the message "The number is not greater than 5" is printed.

The if-else if-else Statement

The if-else if-else statement lets you check multiple conditions. It allows for more than one alternative block of code to be executed depending on the condition that is true.

Syntax:


if (condition1) {
    // code block if condition1 is true
} else if (condition2) {
    // code block if condition2 is true
} else {
    // code block if neither condition1 nor condition2 is true
}

Example:


let number = 8;
if (number > 10) {
    console.log("The number is greater than 10.");
} else if (number === 8) {
    console.log("The number is equal to 8.");
} else {
    console.log("The number is less than 8.");
}

In this case, since number is 8, the second condition is true, and the message "The number is equal to 8" will be printed.

The switch Statement

The switch statement is used to perform multiple comparisons based on the value of a single variable. It is an alternative to using many if-else if statements when comparing the same variable to different values.

Syntax:


switch (expression) {
    case value1:
        // code block if expression === value1
        break;
    case value2:
        // code block if expression === value2
        break;
    default:
        // code block if expression does not match any case
}

Example:


let fruit = "apple";
switch (fruit) {
    case "banana":
        console.log("This is a banana.");
        break;
    case "apple":
        console.log("This is an apple.");
        break;
    case "orange":
        console.log("This is an orange.");
        break;
    default:
        console.log("Unknown fruit.");
}

In this case, since fruit is "apple", the corresponding case block will be executed, and "This is an apple" will be printed.

Ternary (Conditional) Operator

The ternary operator is a shorthand for if-else statements. It is a concise way to write conditional expressions.

Syntax:


condition ? expression1 : expression2;

Example:


let age = 18;
let canDrive = (age >= 16) ? "Yes" : "No";
console.log(canDrive); // Output: Yes

Here, since age is 18, the condition evaluates to true, and the value "Yes" is assigned to canDrive.
