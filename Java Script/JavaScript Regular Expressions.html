JavaScript Regular Expressions

Regular Expression in JavaScript

A regular expression (or regex) is a powerful tool used in JavaScript to search, match, and manipulate strings based on patterns. Regular expressions consist of characters and operators that form search patterns. These patterns can be used to check if a string contains a specific sequence of characters or to find, replace, or validate text.

In JavaScript, regular expressions are implemented using the RegExp object or through the regular expression literal syntax. Regular expressions can be used in a variety of situations, such as form validation, text parsing, and data extraction from strings.

Creating a Regular Expression

In JavaScript, you can create regular expressions in two ways:

  • Regular Expression Literals: Enclosed between forward slashes, /pattern/.
  • RegExp Constructor: Using the new RegExp() syntax.

Example of Regular Expression Literal

let regex = /hello/;

Example using RegExp Constructor

let regex = new RegExp('hello');

Basic Syntax and Symbols

Regular expressions have a set of special characters and syntax that define how patterns are matched. Below are the most commonly used symbols and syntax in JavaScript regular expressions:

Symbol Meaning
. Matches any character except newlines.
\d Matches any digit (0-9).
\w Matches any word character (alphanumeric & underscore).
\s Matches any whitespace character (spaces, tabs, line breaks).
\b Matches a word boundary.
^ Matches the start of a string.
$ Matches the end of a string.
[abc] Matches any of the characters a, b, or c.
[^abc] Matches any character except a, b, or c.
+ Matches one or more of the preceding element.
* Matches zero or more of the preceding element.
? Matches zero or one of the preceding element.
{n} Matches exactly n occurrences of the preceding element.
{n,} Matches n or more occurrences of the preceding element.

Using Regular Expressions in JavaScript

In JavaScript, regular expressions can be used in various methods such as test(), exec(), match(), replace(), and more. Below are examples of using regular expressions with different JavaScript methods.

test() Method

The test() method tests whether a string matches a given regular expression. It returns true if the string matches the pattern, or false otherwise.


let regex = /hello/;
console.log(regex.test('hello world'));  // Output: true
console.log(regex.test('hi there'));     // Output: false

exec() Method

The exec() method searches for a match in a string and returns an array of results. If no match is found, it returns null.


let regex = /hello/;
let result = regex.exec('hello world');
console.log(result);  // Output: ["hello"]

match() Method

The match() method is used to retrieve the result of matching a regular expression against a string. It returns an array of matches or null if no match is found.


let str = 'hello world';
let result = str.match(/world/);
console.log(result);  // Output: ["world"]

replace() Method

The replace() method is used to search for a pattern and replace it with a new string. It can also take a regular expression pattern.


let str = 'hello world';
let newStr = str.replace(/world/, 'JavaScript');
console.log(newStr);  // Output: "hello JavaScript"

Flags in Regular Expressions

Regular expressions in JavaScript can include flags that modify their behavior. The most commonly used flags are:

  • g (global): Matches all occurrences of the pattern in the string, not just the first.
  • i (case-insensitive): Makes the match case-insensitive.
  • m (multiline): Changes the behavior of ^ and $ to match the beginning and end of each line, not just the string.

Example with Flags


let regex = /hello/gi;
let str = 'Hello world, hello world!';
let result = str.match(regex);
console.log(result);  // Output: ["Hello", "hello"]

Regular Expression Groups and Capturing

Groups allow you to group part of a regular expression together, and capturing allows you to extract these groups from the matched string.

Example with Groups


let regex = /(\d{3})-(\d{3})-(\d{4})/;
let str = '123-456-7890';
let result = str.match(regex);
console.log(result);  // Output: ["123-456-7890", "123", "456", "7890"]

The parentheses ( ) are used to define groups. When the regular expression matches, each group can be accessed using its index in the result array.

Best Practices for Using Regular Expressions

  • Keep it Simple: Use the simplest regular expression that meets your needs to ensure readability and maintainability.
  • Test Regular Expressions: Regular expressions can sometimes be complex, so it is a good idea to test them using online tools like regex101 or regexr.
  • Escape Special Characters: If you need to match special characters, remember to escape them with a backslash \.
