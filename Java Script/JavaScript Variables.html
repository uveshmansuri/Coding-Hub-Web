JavaScript Variables

Introduction

Variables are fundamental to programming in JavaScript. They allow developers to store, retrieve, and manipulate data efficiently. A variable is essentially a named container for a value.

Declaring Variables

In JavaScript, variables can be declared using three keywords: var, let, and const. Each serves a unique purpose and has specific characteristics.

Example: Declaring Variables

// Using var (function-scoped)
var name = "John";

// Using let (block-scoped)
let age = 25;

// Using const (block-scoped, immutable)
const country = "India";

Variable Scope

Scope determines where a variable can be accessed in the code. JavaScript has two types of scopes:

  • Global Scope: Variables declared outside any function or block are globally scoped and accessible from anywhere in the code.
  • Local Scope: Variables declared inside a function or block are locally scoped and accessible only within that function or block.

Example: Global and Local Scope

// Global variable
globalVar = "I am global";

function exampleFunction() {
    // Local variable
    let localVar = "I am local";
    console.log(globalVar); // Accessible
    console.log(localVar);  // Accessible
}

console.log(globalVar); // Accessible
console.log(localVar);  // Error: localVar is not defined

var, let, and const: Key Differences

Feature var let const
Scope Function-scoped Block-scoped Block-scoped
Reassignment Allowed Allowed Not Allowed
Hoisting Yes, initialized to undefined Yes, but not initialized Yes, but not initialized

Best Practices for Using Variables

  • Use const by default for values that do not change.
  • Use let for variables that will be reassigned.
  • Avoid using var to prevent scope-related issues.
  • Choose meaningful and descriptive variable names.
  • Follow camelCase convention for variable names (e.g., userName).

Dynamic Typing in JavaScript

JavaScript is dynamically typed, meaning variables can hold values of any data type, and the type can change at runtime.

Example: Dynamic Typing

let dynamicVar = "Hello, World!"; // String
dynamicVar = 42;                // Number
dynamicVar = true;              // Boolean
console.log(dynamicVar);
