JavaScript Data Types

Introduction

JavaScript is a dynamically typed language, which means that variables do not need to declare a type explicitly. The type of a variable is determined based on the value assigned to it. Understanding data types is crucial for writing efficient and bug-free code.

Primitive Data Types

JavaScript has the following primitive data types:

  • String: Represents textual data. Strings are enclosed in single quotes, double quotes, or backticks.
  • Number: Represents both integers and floating-point numbers.
  • Boolean: Represents a logical value, either true or false.
  • Undefined: A variable that has been declared but not assigned a value.
  • Null: Represents the intentional absence of a value.
  • Symbol: A unique and immutable primitive value introduced in ES6.
  • BigInt: Represents integers with arbitrary precision, introduced in ES2020.

Example: Primitive Data Types

let name = "Alice";        // String
let age = 30;              // Number
let isStudent = false;     // Boolean
let job;                   // Undefined
let salary = null;         // Null
let uniqueId = Symbol();   // Symbol
let bigNumber = 12345678901234567890n; // BigInt

console.log(name, age, isStudent, job, salary, uniqueId, bigNumber);

Reference Data Types

Reference data types are objects that are used to store collections of data or more complex entities. The most commonly used reference types are:

  • Object: A collection of key-value pairs.
  • Array: An ordered list of values, which can hold mixed data types.
  • Function: A reusable block of code.
  • Date: Used for working with dates and times.
  • RegExp: Represents regular expressions.

Example: Reference Data Types

let user = {             // Object
    name: "Bob",
    age: 28
};

let scores = [90, 85, 88]; // Array

function greet() {        // Function
    return "Hello!";
}

console.log(user, scores, greet());

Type Conversion

JavaScript provides mechanisms to convert data from one type to another, either explicitly or implicitly.

Explicit Conversion

Using functions like String(), Number(), or Boolean() to convert data types explicitly.

let num = 42;
let str = String(num); // Explicit conversion to string
let bool = Boolean(num); // Explicit conversion to boolean

console.log(typeof str, typeof bool);

Implicit Conversion

Occurs when JavaScript automatically converts data types during operations.

let result = "5" + 2; // Implicitly converts 2 to "2"
console.log(result);   // Outputs: "52"

let sum = "5" * 2;    // Implicitly converts "5" to 5
console.log(sum);      // Outputs: 10

Checking Data Types

You can use the typeof operator to determine the type of a variable.

let value = "Hello";
console.log(typeof value); // Outputs: string

value = 42;
console.log(typeof value); // Outputs: number

value = null;
console.log(typeof value); // Outputs: object (this is a known JavaScript quirk)
