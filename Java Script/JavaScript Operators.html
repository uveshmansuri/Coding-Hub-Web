Operators in JavaScript

1. Arithmetic Operators

Arithmetic operators are used to perform mathematical operations such as addition, subtraction, etc.

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • % : Modulus (remainder)
  • ++ : Increment
  • -- : Decrement

Example:


let a = 10, b = 5;
let sum = a + b; // sum = 15
let difference = a - b; // difference = 5

2. Assignment Operators

Assignment operators are used to assign values to variables.

  • = : Assigns the value on the right to the variable on the left
  • += : Adds and assigns the value
  • -= : Subtracts and assigns the value
  • *= : Multiplies and assigns the value
  • /= : Divides and assigns the value
  • %= : Modulo and assigns the value

Example:


let x = 10;
x += 5; // x = 15
x *= 2; // x = 30

3. Comparison Operators

Comparison operators are used to compare two values. They return a Boolean value (true or false).

  • == : Equal to
  • === : Strictly equal to (compares both value and type)
  • != : Not equal to
  • !== : Strictly not equal to
  • > : Greater than
  • < : Less than
  • >= : Greater than or equal to
  • <= : Less than or equal to

Example:


let a = 5, b = '5';
console.log(a == b); // true (loose equality)
console.log(a === b); // false (strict equality)

4. Logical Operators

Logical operators are used to perform logical operations.

  • && : Logical AND
  • || : Logical OR
  • ! : Logical NOT

Example:


let a = true, b = false;
console.log(a && b); // false
console.log(a || b); // true
console.log(!a); // false

5. Conditional (Ternary) Operator

The ternary operator is a shorthand for the if-else statement.

  • condition ? expr1 : expr2

Example:


let age = 18;
let canVote = age >= 18 ? "Yes" : "No";
console.log(canVote); // "Yes"

6. Typeof Operator

The typeof operator is used to check the type of a variable.

Example:


let a = 10;
console.log(typeof a); // "number"

let str = "Hello";
console.log(typeof str); // "string"

7. Bitwise Operators

Bitwise operators perform operations on binary representations of numbers.

  • & : AND
  • | : OR
  • ^ : XOR
  • ~ : NOT
  • << : Left shift
  • >> : Right shift
  • >>> : Unsigned right shift

Example:


let x = 5, y = 3;
console.log(x & y); // 1
console.log(x | y); // 7
