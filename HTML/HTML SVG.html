SVG

SVG (Scalable Vector Graphics) is an XML-based format for vector graphics. It allows you to create high-quality graphics that are scalable without losing resolution. SVG is widely supported by modern browsers and is ideal for creating interactive graphics, illustrations, logos, and animations.

Basic Structure of SVG

SVG is embedded within an HTML document using the <svg> element. Within this element, you can define shapes like rectangles, circles, polygons, and paths, as well as style them with CSS or JavaScript.

Example: A Simple SVG Shape

Here is an example of an SVG rectangle:

            <svg width="200" height="200">
                <rect x="50" y="50" width="150" height="100" fill="blue" />
            </svg>

Common SVG Shapes

SVG supports various shapes and graphical elements:

  • Rectangle (<rect>): Used to create rectangles.
  • Circle (<circle>): Creates circles.
  • Ellipse (<ellipse>): Used for ellipses.
  • Line (<line>): Draws straight lines between two points.
  • Polygon (<polygon>): Draws a closed shape with multiple sides.
  • Path (<path>): Allows you to draw complex shapes and curves.

Example: Drawing a Circle

Here's an example of drawing a circle with SVG:

            <svg width="200" height="200">
                <circle cx="100" cy="100" r="50" fill="red" />
            </svg>

SVG Path Element

The <path> element is the most versatile element in SVG. It allows you to draw complex shapes by specifying a series of commands. The d attribute defines the path's instructions.

Example: Using Path to Draw a Heart

This example demonstrates how to draw a heart shape using the <path> element:

            <svg width="200" height="200">
                <path d="M 10,30 
                         C 40,5 65,5 90,30 
                         C 115,55 145,55 170,30 
                         C 195,5 220,5 250,30 
                         C 270,50 270,80 250,110 
                         C 230,140 190,160 170,150
                         C 150,140 120,110 90,100
                         C 60,90 30,110 10,130 Z"
                      fill="pink" />
            </svg>

SVG with CSS Styling

Just like HTML elements, SVG shapes can be styled using CSS. You can style SVG directly using the style attribute or use external stylesheets.

Example: Styling SVG with CSS

            <svg width="200" height="200">
                <rect x="50" y="50" width="100" height="100" class="myRect" />
            </svg>
    
            <style>
                .myRect {
                    fill: green;
                    stroke: black;
                    stroke-width: 5;
                }
            </style>

Animating SVG with CSS

SVG elements can also be animated using CSS animations. You can animate properties like fill color, stroke width, and transform properties.

Example: Animating an SVG Shape

            <svg width="200" height="200">
                <circle cx="100" cy="100" r="50" class="animateCircle" />
            </svg>
    
            <style>
                .animateCircle {
                    fill: blue;
                    transition: fill 0.5s ease;
                }
    
                .animateCircle:hover {
                    fill: orange;
                }
            </style>

SVG Filters

SVG filters allow you to apply graphical effects such as blur, shadows, or lighting to your shapes.

Example: Applying a Blur Filter

            <svg width="200" height="200">
                <defs>
                    <filter id="blur-effect" x="0" y="0" width="150%" height="150%">
                        <feGaussianBlur in="SourceGraphic" stdDeviation="5" />
                    </filter>
                </defs>
                <circle cx="100" cy="100" r="50" fill="blue" filter="url(#blur-effect)" />
            </svg>
