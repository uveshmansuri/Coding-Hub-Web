HTML Tables

HTML tables are used to organize data in rows and columns, making it easy to present structured data such as schedules, reports, or comparisons. Tables are defined using the <table> tag, and rows and cells are defined with <tr>, <th>, and <td> tags.

Basic Structure of an HTML Table

Here is the basic syntax of an HTML table:

<table>
    <tr> 
        <th>Header 1</th> 
        <th>Header 2</th>
    </tr>
    <tr> 
        <td>Data 1</td> 
        <td>Data 2</td>
    </tr>
</table>

Rendered Output:

Header 1 Header 2
Data 1 Data 2

Table Elements Explained

  • <table>: The main container that defines the table.
  • <tr>: Defines a row in the table.
  • <th>: Defines a header cell, which is bold and centered by default.
  • <td>: Defines a data cell that contains table content.

Adding Borders, Padding, and Alignment

You can enhance the appearance of tables using CSS to add borders, padding, and alignment:

table {
    border-collapse: collapse; /* Removes double borders */
    width: 100%;              /* Makes the table full width */
}

th, td {
    border: 1px solid #ddd;   /* Adds borders to cells */
    padding: 8px;             /* Adds padding for better readability */
    text-align: left;         /* Aligns text to the left */
}

th {
    background-color: #0078d7; /* Adds a background color to header cells */
    color: white;              /* Changes header text color to white */
}

Advanced Features

Merging Cells Using colspan and rowspan

You can merge cells horizontally or vertically using the colspan and rowspan attributes.

Example:

<table>
    <tr>
        <th>Name</th>
        <th colspan="2">Details</th> 
    </tr>
    <tr>
        <td>Uvesh</td>
        <td rowspan="2">Age: 25</td> 
        <td>City: Bharuch</td>
    </tr>
    <tr>
        <td>Sam</td>
        <td>City: Surat</td>
    </tr>
</table>

Rendered Output:

Name Details
Uvesh Age: 19 City: Bharuch
Sam City: Suart

Adding a Caption

You can add a title or description to your table using the <caption> tag.

Example:

<table>
    <caption>Student Grades</caption> 
    <tr>
        <th>Name</th>
        <th>Subject</th>
        <th>Grade</th>
    </tr>
    <tr>
        <td>Uvesh</td>
        <td>Math</td>
        <td>A</td>
    </tr>
    <tr>
        <td>Muskan</td>
        <td>Science</td>
        <td>B</td>
    </tr>
</table>

Rendered Output:

Student Grades
Name Subject Grade
Uvesh Math A
Muskan Science B
