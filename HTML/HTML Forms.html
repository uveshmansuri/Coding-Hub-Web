Creating Forms in HTML

What is an HTML Form?

An HTML form is a structure used to collect user inputs such as text, selections, or choices, and submit it to a server for processing. Forms consist of input fields, buttons, and other elements enclosed within the <form> tag.

Basic Form Structure

Here is a simple example of a form structure:

<form action="submit_page.html" method="POST">
    <label for="username">Username:</label>
    <input type="text" id="username" name="user_name" required>

    <label for="email">Email:</label>
    <input type="email" id="email" name="user_email" required>

    <button type="submit">Submit</button>
</form>

Important Form Attributes

  • action: Specifies the URL where form data will be sent.
  • method: Defines the HTTP method to send the form data (GET or POST).
  • name: Identifies the input field for server-side processing.
  • required: Makes a field mandatory to fill out.

Explanation of Common Input Types

  • Text: Collects short text input (e.g., name).
  • Email: Ensures valid email format.
  • Password: Masks the input characters.
  • Radio: Allows single selection from multiple options.
  • Checkbox: Allows multiple selections.
  • Textarea: Collects longer text inputs.
  • Dropdown: Provides a list of options to choose from.

Form Validation

HTML5 provides built-in form validation features that help ensure data is correctly formatted before submission. Some common validation attributes include:

  • required: Ensures that the field is filled out before submitting the form.
  • pattern: Specifies a regular expression to match for input validation (e.g., phone numbers, zip codes).
  • minlength / maxlength: Specifies the minimum and maximum number of characters allowed in the input field.
  • type: Helps validate the type of input. For example, type="email" will ensure the input is a valid email address format.

Example of Form Validation

Here is an example of a form that uses validation:

<form id="contactForm" action="#" method="POST" onsubmit="return validateForm()">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username" required>
    <br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    <br>
    <label for="phone">Phone Number:</label>
    <input type="text" id="phone" name="phone" pattern="^\d{10}$" title="Phone number must be 10 digits" required>
    <br>
    <input type="submit" value="Submit">
</form>

JavaScript Form Validation

Sometimes, HTML's built-in validation might not be sufficient for complex scenarios, and additional validation can be done using JavaScript.

Here is a simple JavaScript example that validates the form:

<script>
    function validateForm() {
        var username = document.getElementById("username").value;
        var email = document.getElementById("email").value;
        var phone = document.getElementById("phone").value;
        var errorMessage = document.getElementById("error-message")
        // Validate phone number
        var phonePattern = /^\d{10}$/;
        if (!phonePattern.test(phone)) {
            errorMessage.textContent = "Phone number must be exactly 10 digits.";
            return false;
        
        // Validate username
        if (username == "") {
            errorMessage.textContent = "Username is required.";
            return false;
        
        // Validate email
        if (email == "") {
            errorMessage.textContent = "Email is required.";
            return false;
        
        // If all validations pass
        errorMessage.textContent = "";
        return true;
    }
</script>
