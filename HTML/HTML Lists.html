HTML Lists

What are HTML Lists?

HTML lists are used to group related items together in a structured format. They make content more organized and readable. There are two main types of lists:

  • Ordered Lists (<ol>)
  • Unordered Lists (<ul>)

1. Ordered List (<ol>)

An ordered list is used when the order of items matters. The items are usually numbered (default) or can use other markers such as Roman numerals or alphabets.

Example:

<ol>
    <li>Step 1: Install HTML Editor</li>
    <li>Step 2: Write Code</li>
    <li>Step 3: Test the Code</li>
</ol>

Rendered Output:

  1. Step 1: Install HTML Editor
  2. Step 2: Write Code
  3. Step 3: Test the Code

2. Unordered List (<ul>)

An unordered list is used when the order of items is not important. Items are marked with bullets by default.

Example:

<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>

Rendered Output:

  • HTML
  • CSS
  • JavaScript

Key Differences:

  • Ordered List: Used for sequential or ranked data.
  • Unordered List: Used for non-sequential or unranked data.
