HTML Input Types

Introduction

HTML provides various input types to handle different kinds of user inputs. Below is a detailed explanation and examples of common input types.

Common Input Types

Type: text — Used for single-line text input.

Type: password — Masks the input for sensitive information like passwords.

Type: email — Ensures the entered value is a valid email address.

Type: number — Used for numeric input with optional range limits.

Type: date — Allows selection of a specific date.

Type: time — Allows input of a specific time.

Type: file — Used to upload files.

Type: color — Allows selection of a color.

Type: range — Displays a slider to select a value within a range.

Type: search — Used for search input fields.

Type: url — Ensures the entered value is a valid URL.

Type: tel — Used for entering phone numbers.

Type: submit — Submits the form data.
