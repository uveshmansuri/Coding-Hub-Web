Adding Images in HTML

Introduction

Images are an essential part of modern web design. They help convey information visually, enhance the user experience, and make websites more engaging. HTML provides several ways to add images to a webpage using the <img> tag.

The <img> Tag

The <img> tag is used to embed images in an HTML document. It is an empty tag, meaning it does not have a closing tag. The basic syntax is:

Attributes of the <img> Tag

  • src: Specifies the path to the image file. This can be a relative or absolute URL.
  • alt: Provides alternative text if the image cannot be displayed. This is important for accessibility.
  • width: Specifies the width of the image in pixels or as a percentage.
  • height: Specifies the height of the image in pixels or as a percentage.

Examples

Example 1: Adding a Basic Image

<img src="image.jpg" alt="A beautiful scenery" width="500">

This example displays an image with a specified width of 500 pixels.

Example 2: Adding an Image with a Relative Path

To add an image stored in the same folder as your HTML file, use a relative path:

<img src="images/photo.jpg" alt="Photo from folder">

Example 3: Adding an Image with a Link

Images can be wrapped in links to make them clickable:

<a href="https://example.com"><img src="image.jpg" alt="Clickable image"></a>

Responsive Images

To make images responsive and adapt to different screen sizes, use CSS:

img { max-width: 100%; height: auto; }

This ensures the image scales proportionally to its container.

Advanced Techniques

Using <picture> for Responsive Images

The <picture> element allows you to define multiple image sources for different screen sizes:

<picture> <source srcset="image-large.jpg" media="(min-width: 1024px)"> <source srcset="image-small.jpg" media="(max-width: 768px)"> <img src="fallback.jpg" alt="Responsive image"> </picture>

This helps optimize loading by serving appropriately sized images based on the user's device.
