Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create web pages and web applications. HTML structures the content of a website by using various tags and elements.

The term "HyperText" refers to the ability to link documents together, enabling navigation between pages via hyperlinks. "Markup Language" means that HTML uses tags to define elements within the document.

Basic Structure of an HTML Document

Every HTML document follows a specific structure. Here's the basic template:

<!DOCTYPE html> 
<html> 
    <head> 
        <title>Page Title</title> 
    </head>
    <body> 
        <h1>Welcome to HTML</h1>
        <p>This is a simple paragraph.</p>
    </body>
</html>

Key Features of HTML

HTML is a foundational technology for the web. Some of its key features include:

  • Tag-based Structure: HTML uses tags like <h1>, <p>, and <div> to structure content.
  • Attributes: Tags can have attributes (e.g., class, id, src) to provide additional properties or styling hooks.
  • Browser Compatibility: HTML works across all modern web browsers.
  • Interoperability: HTML works seamlessly with CSS for styling and JavaScript for interactivity.

HTML Tags and Elements

HTML documents are made up of various tags and elements. Here's an overview:

  • Headings: Used to define titles and subtitles. Tags range from <h1> (most important) to <h6> (least important).
  • Paragraphs: The <p> tag defines blocks of text.
  • Links: The <a> tag creates hyperlinks to other pages or resources. Example: <a href="https://example.com">Visit Example</a>.
  • Images: The <img> tag embeds images. Example: <img src="image.jpg" alt="Description">.
  • Lists:
    • <ul>: Unordered list (bullets).
    • <ol>: Ordered list (numbers).
    • <li>: List item.

HTML Attributes

Attributes provide additional information about elements. Common attributes include:

  • class: Used to assign a CSS class for styling.
  • id: Assigns a unique identifier to an element.
  • href: Specifies the URL for a link.
  • src: Specifies the source file for media elements like images and videos.
  • alt: Provides alternative text for images.

Example: A Simple HTML Page

Here's a complete example of a basic HTML page:

<!DOCTYPE html>
<html>
    <head>
        <title>My First Web Page</title>
    </head>
    <body>
        <h1>Welcome to My Website</h1>
        <p>This is a paragraph about me.</p>
        <a href="https://example.com">Visit Example</a>
        <img src="profile.jpg" alt="My Profile Photo">
    </body>
</html>
