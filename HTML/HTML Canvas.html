HTML Canvas

The HTML <canvas> element is a container used to draw graphics, such as lines, shapes, text, and images, directly on a web page. It provides a versatile way to create dynamic and interactive visual content. The drawing is accomplished using JavaScript and the Canvas API.

How to Use Canvas

To use the <canvas> element, you need to define its width and height, and then use JavaScript to access its drawing context. The context provides methods to draw shapes, lines, text, and images on the canvas.

Here is an example of a simple canvas with a red rectangle:

<canvas id="myCanvas" width="100%" height="200" style="border:1px solid #000;"></canvas>

<script>
// Access the canvas element and its context
var canvas = document.getElementById('myCanvas');
var ctx = canvas.getContext('2d');

// Draw a rectangle
ctx.fillStyle = 'red';
ctx.fillRect(50, 50, 100, 70);
</script>

Drawing on the Canvas

The canvas element provides various methods to draw shapes and lines. Below are some of the most commonly used methods:

  • fillRect(x, y, width, height): Draws a filled rectangle.
  • strokeRect(x, y, width, height): Draws the outline of a rectangle.
  • beginPath(): Starts a new path for drawing shapes.
  • moveTo(x, y): Moves the "pen" to a specific coordinate.
  • lineTo(x, y): Draws a line to the specified coordinates.
  • arc(x, y, radius, startAngle, endAngle): Draws an arc or circle.
  • fill(): Fills the current path with the current fill style.
  • stroke(): Strokes the current path with the current stroke style.

Example: Drawing a Circle

Here is an example where we draw a circle on the canvas:

Output

<canvas id="circleCanvas" width="auto" height="auto"></canvas>
<script>
    var circleCanvas = document.getElementById('circleCanvas');
    var ctx2 = circleCanvas.getContext('2d');
    
    // Draw a circle
    ctx2.beginPath();
    ctx2.arc(50, 50, 50, 0, 2 * Math.PI);
    ctx2.fillStyle = 'blue';
    ctx2.fill();
    ctx2.strokeStyle = 'black';
    ctx2.stroke();
</script>

Canvas Events

You can interact with the canvas using events like mouse clicks, movements, and key presses. Here is an example of how to draw shapes interactively on the canvas:

<canvas id="drawCanvas" width="auto" height="auto" style="border:1px solid #000;"></canvas>
<script>
    var drawCanvas = document.getElementById('drawCanvas');
    var ctx3 = drawCanvas.getContext('2d');
    var drawing = false;
    // Start drawing when mouse is pressed
    drawCanvas.addEventListener('mousedown', function(event) {
        drawing = true;
        ctx3.beginPath();
        ctx3.moveTo(event.offsetX, event.offsetY);
    });
    // Draw while mouse is moving
    drawCanvas.addEventListener('mousemove', function(event) {
        if (drawing) {
            ctx3.lineTo(event.offsetX, event.offsetY);
            ctx3.stroke();
        }
    });
    // Stop drawing when mouse is released
    drawCanvas.addEventListener('mouseup', function() {
        drawing = false;
    });
    // Stop drawing if mouse leaves canvas
    drawCanvas.addEventListener('mouseout', function() {
        drawing = false;
    });
</script>
