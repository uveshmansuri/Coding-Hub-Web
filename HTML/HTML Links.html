HTML Links

Introduction

Links in HTML, also known as **hyperlinks**, allow users to navigate between different pages, sections, or even websites. They are created using the <a> (anchor) tag.

Basic Syntax

The basic syntax for creating a hyperlink is:

<a href="URL">Link Text</a>

Where:

  • href: Specifies the URL or location the link points to.
  • Link Text: The clickable text displayed on the webpage.

Types of Links

HTML links can point to various destinations:

1. Links to External Websites

<a href="https://www.google.com">Visit Google</a>

Example: Visit Google

2. Links to Internal Pages

<a href="index.html">Coding Hub</a>

Example: Coding Hub

3. Links to Specific Sections (Anchor Links)

Anchor links navigate to a specific section within the same page or another page.

<a href="#section1">Go to Section 1</a>

<h2 id="section1">Section 1</h2>

Example: Go to Example Section

4. Email Links

Use the mailto: prefix to create an email link.

<a href="mailto:example@example.com">Email Us</a>

Example: Email Us

5. Links to Download Files

Provide a direct link to a file for users to download.

<a href="../Images/fc.jpeg" download>Download Image</a>

Example: Download Image

Attributes of the <a> Tag

Additional attributes can modify the behavior of links:

  • target: Specifies where to open the linked document.
    • _self (default): Opens the link in the same tab.
    • _blank: Opens the link in a new tab.
    • _parent: Opens the link in the parent frame.
    • _top: Opens the link in the full body of the window.
  • rel: Specifies the relationship between the current page and the linked page (e.g., nofollow, noopener).

Example:

<a href="https://example.com" target="_blank" rel="noopener">Visit Example</a>

Practical Examples

Here’s how different types of links look:

  • External Link: Visit Wikipedia
  • Internal Link: Coding Hub
  • Anchor Link: Back to Top
  • Email Link: Contact Support
  • File Download: Download Image
