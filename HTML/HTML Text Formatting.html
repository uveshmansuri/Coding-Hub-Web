Text Formatting in HTML

Introduction

HTML provides various tags for text formatting, enabling you to style, emphasize, and structure your content effectively. These tags are used for emphasis, strong importance, quoting, code display, and more.

Common Text Formatting Tags

Below is a list of commonly used text formatting tags in HTML:

  • <b>: Makes text bold.
  • <strong>: Indicates strong importance (renders as bold).
  • <i>: Makes text italic.
  • <em>: Indicates emphasis (renders as italic).
  • <mark>: Highlights text.
  • <small>: Displays smaller text.
  • <del>: Displays deleted text with a strikethrough.
  • <ins>: Displays inserted text (underlined).
  • <sub>: Displays subscript text.
  • <sup>: Displays superscript text.
  • <code>: Formats text as code.
  • <q>: Adds inline quotations.
  • <blockquote>: Adds block-level quotations.

Examples of Text Formatting

Here are some examples of how text formatting tags are used:

<!-- Bold and Italic Text -->
This is <b>bold text</b>.
This is <i>italic text</i>.

<!-- Emphasis and Strong Importance -->
This is <strong>important text</strong>.
This is <em>emphasized text</em>.

<!-- Highlighted and Small Text -->
This is <mark>highlighted text</mark>.
This is <small>small text</small>.

<!-- Deleted and Inserted Text -->
This is <del>deleted text</del>.
This is <ins>inserted text</ins>.

<!-- Superscript and Subscript Text -->
This is E = MC<sup>2</sup>.
This is H<sub>2</sub>O.

<!-- Code and Quotes -->
This is <code>inline code</code>.
This is <q>an inline quotation</q>.
