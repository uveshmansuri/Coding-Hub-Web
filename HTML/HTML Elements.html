HTML Elements and Attributes

What are HTML Elements?

An HTML element is the building block of HTML. It consists of a start tag, content, and an end tag.

The general syntax of an element is:

<tagname>Content</tagname>

For example:

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

Types of HTML Elements

  • Block-Level Elements: These elements start on a new line and take up the full width of the container. Examples include:
    • <div>
    • <p>
    • <h1> to <h6>
  • Inline Elements: These elements do not start on a new line and only take up as much width as necessary. Examples include:
    • <span>
    • <a>
    • <img>
  • Void Elements: These elements do not have closing tags. Examples include:
    • <br> (line break)
    • <img> (image)
    • <input> (form input)

What are HTML Attributes?

An attribute provides additional information about an HTML element. Attributes are always specified in the opening tag and come in name-value pairs like this:

<tagname attribute="value">Content</tagname>

For example:

<a href="https://example.com">Visit Example</a>

Commonly Used HTML Attributes

  • href: Specifies the URL for a link (used with <a> tags).
  • src: Specifies the source file for media (used with <img> and <video> tags).
  • alt: Provides alternative text for images.
  • id: Assigns a unique identifier to an element.
  • class: Groups elements for CSS styling.
  • style: Adds inline CSS to an element.
  • target: Specifies how to open a link (e.g., "_blank" for a new tab).

Example with Multiple Attributes

<img src="example.jpg" alt="An example image" width="300" height="200">

HTML Nested Elements

HTML allows nesting one element inside another to create a hierarchy. Example:

<div>
    <h1>Welcome to My Page</h1>
    <p>This is a paragraph inside a <strong>div</strong> element.</p>
</div>

HTML Global Attributes

Global attributes can be used with any HTML element. Some common global attributes include:

  • id: Specifies a unique identifier for an element.
  • class: Assigns one or more class names for styling.
  • style: Adds inline styles to an element.
  • title: Provides additional information when hovered over.
  • lang: Declares the language of the element’s content.

Example:

<p id="intro" class="highlight" style="color: blue;" title="Introduction">
    This is an introductory paragraph.
</p>
