HTML Styles Overview

What are HTML Styles?

HTML styles allow you to control the appearance of elements in your webpage. Styling can be achieved through three main methods:

  • Inline Styles: Apply styles directly to an element using the style attribute.
  • Internal CSS: Use the <style> element within the <head> section of your HTML document.
  • External CSS: Link to an external stylesheet using the <link> element, which keeps the style separate from the HTML content.

Inline Styles

Inline styles are used to apply CSS directly within an HTML tag using the style attribute.

This is a paragraph with inline styling.

Internal CSS

Internal CSS is defined within the <style> element, which is placed in the <head> section of the HTML document. It is best for small to medium-sized projects where styling is applied to a single page.

External CSS

External CSS is when the CSS rules are placed in a separate file with a .css extension. The external file is linked to the HTML file using the <link> element. This is the most efficient way to style multiple pages of a website.

For example, an external CSS file would look like this:

<link rel="stylesheet" href="styles.css">

CSS Selectors

CSS selectors are patterns used to select the elements to apply styles. Here are the common types:

  • Element Selector: Targets all elements of a given type (e.g., p, h1). Example: p { color: blue; }
  • Class Selector: Targets all elements with a specific class (e.g., .highlight). Example: .highlight { background-color: yellow; }
  • ID Selector: Targets a specific element with a given ID (e.g., #important-text). Example: #important-text { font-size: 20px; }
  • Attribute Selector: Targets elements based on their attributes (e.g., input[type="text"]).
