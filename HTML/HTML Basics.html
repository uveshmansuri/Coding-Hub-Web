HTML Basics

What is HTML?

HTML, or HyperText Markup Language, is the foundation of web development. It is a markup language used to structure and display content on the web. HTML is not a programming language but a descriptive language that uses tags to define elements.

It works alongside CSS (for styling) and JavaScript (for interactivity) to create dynamic, user-friendly web pages.

Key Features of HTML

  • Provides structure and layout to web pages.
  • Supports multimedia elements like images, videos, and audio.
  • Works across all browsers and devices.
  • Extensible with frameworks like Bootstrap for advanced features.

Basic HTML Document Structure

An HTML document starts with a declaration and is divided into two main parts: head and body.

<!DOCTYPE html> 
<html> 
    <head>
        <title>My First Web Page</title> 
        <meta charset="UTF-8"> 
    </head>
    <body> 
        <h1>Hello, World!</h1>
        <p>This is a simple HTML document.</p>
    </body>
</html>

Common HTML Tags

HTML uses a variety of tags to define elements. Here are some commonly used ones:

  • <h1> to <h6>: Defines headings, with <h1> being the largest and <h6> the smallest.
  • <p>: Defines a paragraph.
  • <a>: Creates hyperlinks. Example: 
    <a href="https://example.com">Click Here</a>
  • <img>: Embeds images. Example: 
    <img src="image.jpg" alt="Description">
  • <ul> and <ol>: Define unordered and ordered lists. Example: 
    <ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

HTML Attributes

Attributes add extra information to elements. They are written within the opening tag and usually come in key-value pairs.

  • href: Specifies the URL of a link (used with <a>).
  • src: Specifies the source file (used with <img>, <video>, etc.).
  • alt: Provides alternative text for an image.
  • class: Assigns a class name for styling with CSS.
  • id: Assigns a unique identifier to an element.
Example:
<a href="https://example.com" target="_blank">Visit Example</a>

HTML Media Elements

HTML supports various media elements, such as:

  • <audio>: Embeds audio files. Example: 
    <audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>
  • <video>: Embeds video files. Example: 
    <video controls width="640" height="360">
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video element.
</video>

Semantic HTML

Semantic HTML introduces meaningful tags to improve accessibility and SEO. Examples include:

  • <header>: Defines the header section.
  • <nav>: Defines a navigation bar.
  • <article>: Defines an article or piece of content.
  • <footer>: Defines the footer section.

Example:

<header>
    <h1>Welcome to My Blog</h1>
</header>
<nav>
    <a href="#home">Home</a> | <a href="#about">About</a>
</nav>
<article>
    <h2>Blog Post Title</h2>
    <p>This is the content of the blog post.</p>
</article>
<footer>
    <p>© 2024 My Blog. All rights reserved.</p>
</footer>

HTML Forms

Forms are used to collect user input. Example:

<form action="submit.php" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name"><br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br>
    <button type="submit">Submit</button>
</form>
