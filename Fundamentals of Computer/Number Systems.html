Number Systems and Conversions

Number systems are methods of representing numbers using distinct symbols. These systems are essential in both theoretical mathematics and practical computer applications. The four primary number systems are:

  • Binary: Base 2 (0 and 1).
  • Octal: Base 8 (0 to 7).
  • Decimal: Base 10 (0 to 9).
  • Hexadecimal: Base 16 (0 to 9 and A to F).

1. Binary Number System

The binary number system is the foundation of computer systems. It uses two digits: 0 and 1, corresponding to off and on states in electronic circuits.

  • Base: 2
  • Digits: 0, 1
  • Example: 10102

Applications:

  • Digital electronics and computer systems.
  • Data representation in machine-level programming.

Example Conversion: Binary to Decimal

Convert 10102 to decimal:
Step 1: Write the binary number and its positional values:
(1 × 23) + (0 × 22) + (1 × 21) + (0 × 20)
Step 2: Calculate each term:
= (1 × 8) + (0 × 4) + (1 × 2) + (0 × 1)
Step 3: Add the results:
= 8 + 0 + 2 + 0 = 1010

Example Conversion: Decimal to Binary

Convert 2510 to binary:
Step 1: Divide by 2 and record the remainders:
25 ÷ 2 = 12 remainder 1
12 ÷ 2 = 6 remainder 0
6 ÷ 2 = 3 remainder 0
3 ÷ 2 = 1 remainder 1
1 ÷ 2 = 0 remainder 1
Step 2: Write the remainders in reverse order:
2510 = 110012

2. Octal Number System

The octal number system uses eight digits, from 0 to 7. It is often used as a shorthand for binary numbers because each octal digit represents three binary bits.

  • Base: 8
  • Digits Used: 0 to 7
  • Example: 178

Applications:

  • Efficient representation of binary numbers.
  • Used in computer programming for permissions (e.g., UNIX file permissions).

Example Conversion: Octal to Decimal

Convert 578 to decimal:
Step 1: Write the number with its positional values:
(5 × 81) + (7 × 80)
Step 2: Calculate each term:
= (5 × 8) + (7 × 1)
Step 3: Add the results:
= 40 + 7 = 4710

3. Decimal Number System

The decimal number system is the most familiar to humans, as it is used in everyday life. It uses ten digits, from 0 to 9.

  • Base: 10
  • Digits Used: 0 to 9
  • Example: 34510

Applications:

  • Human-centric calculations and daily life use.
  • Serves as an intermediary in programming for conversions.

4. Hexadecimal Number System

The hexadecimal number system uses 16 digits, from 0 to 9 and A to F (where A represents 10, B represents 11, and so on). It is widely used in computing for memory addressing and color codes.

  • Base: 16
  • Digits Used: 0 to 9, A to F
  • Example: 1A316

Applications:

  • Memory addresses and assembly-level programming.
  • Color representation in web design (e.g., #FFFFFF for white).

Example Conversion: Hexadecimal to Decimal

Convert 2A16 to decimal:
Step 1: Write the number with its positional values:
(2 × 161) + (10 × 160)
Step 2: Calculate each term:
= (2 × 16) + (10 × 1)
Step 3: Add the results:
= 32 + 10 = 4210

Example Conversion: Binary to Hexadecimal

Convert 101101112 to hexadecimal:
Step 1: Group binary digits in sets of 4 (from right):
10110111 → 1011 0111
Step 2: Convert each group to hexadecimal:
1011 = B, 0111 = 7
Step 3: Combine the results:
101101112 = B716

Conclusion

Number systems are vital for data representation and computer operations. Understanding conversions between these systems enhances computational skills and programming efficiency.
