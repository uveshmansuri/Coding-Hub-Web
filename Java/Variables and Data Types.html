Variables and Data Types in Java

Introduction

Variables and data types are the foundation of programming in Java. They enable developers to store, manipulate, and process data efficiently. Understanding these concepts is essential to mastering Java and developing robust applications.

Variables

A variable in Java is a named memory location used to store data temporarily. Each variable has a type that determines the size and layout of its memory, the range of values it can hold, and the operations that can be performed on it.

Declaration and Initialization

  • Declaration: Declaring a variable means specifying its type and name. For example: int age;
  • Initialization: Assigning an initial value to a variable. For example: age = 25; 
Example:
int age; // Declaration
age = 25; // Initialization
int salary = 50000; // Declaration and Initialization

Types of Variables in Java

Java supports three types of variables:

  1. Local Variables:

    Declared inside methods, constructors, or blocks. These variables are created when the method is called and destroyed after the method execution ends. 

    Example:
void display() {
    int num = 10; // Local variable
    System.out.println(num);
}
  2. Instance Variables:

    Declared inside a class but outside any method. They are associated with objects and are initialized to default values if not explicitly initialized. 

    Example:
class Employee {
    String name; // Instance variable
    int age;
}
  3. Static Variables:

    Declared using the static keyword. These variables are shared among all instances of a class and belong to the class, not individual objects. 

    Example:
class Company {
    static String companyName = "Tech Corp"; // Static variable
}

Data Types

Data types specify the type of data a variable can hold. Java is a strongly-typed language, which means every variable must have a defined type.

Categories of Data Types

  • Primitive Data Types: These are predefined by the language and serve as the building blocks for data manipulation.
    • byte
    • short
    • int
    • long
    • float
    • double
    • char
    • boolean
  • Non-Primitive Data Types: These include classes, interfaces, and arrays.
    • String
    • Array
    • Object

1.Primitive Data Types

Details of Primitive Data Types:

Data Type Size Default Value Example
byte 1 byte 0 byte b = 100;
short 2 bytes 0 short s = 5000;
int 4 bytes 0 int i = 100000;
long 8 bytes 0L long l = 15000000000L;
float 4 bytes 0.0f float f = 5.75f;
double 8 bytes 0.0d double d = 19.99;
char 2 bytes '\u0000' char c = 'A';
boolean 1 bit false boolean isTrue = true;

2.Non-Primitive Data Types

Non-primitive data types are also called reference types. Unlike primitive types, these do not store values directly; instead, they store references to the memory location where the actual data is stored.

Common Non-Primitive Data Types:

  • String: Used to store a sequence of characters. Strings are immutable in Java. 
    Example:
String name = "John";
System.out.println(name);
  • Array: A collection of variables of the same type. 
    Example:
int[] numbers = {1, 2, 3, 4, 5};
System.out.println(numbers[0]); // Outputs 1
  • Object: Instances of classes. 
    Example:
class Car {
    String brand;
    int speed;
}
Car myCar = new Car();
myCar.brand = "Toyota";
myCar.speed = 120;

Scope of Variables

In Java, the scope of a variable refers to the part of the program where the variable is accessible. Java has different types of variable scopes based on where they are declared:

1. Local Variables

Local variables are declared inside methods, constructors, or blocks. They are only accessible within the method, constructor, or block where they are declared. Once the method or block ends, the local variable is destroyed, and it cannot be accessed outside its scope.

Example:

public class LocalVariableExample {
    public void exampleMethod() {
        int localVariable = 10;  // Local variable
        System.out.println(localVariable);  // Accessible here
    }

    public static void main(String[] args) {
        LocalVariableExample obj = new LocalVariableExample();
        obj.exampleMethod();
        // System.out.println(localVariable);  // Error! localVariable is not accessible here
    }
}

2. Instance Variables

Instance variables are declared within a class, but outside any method or constructor. They belong to an object of the class and are accessible throughout the class. Each object of the class has its own copy of instance variables.

Example:

public class InstanceVariableExample {
    int instanceVariable = 5;  // Instance variable

    public void displayValue() {
        System.out.println(instanceVariable);  // Accessible within the class
    }

    public static void main(String[] args) {
        InstanceVariableExample obj1 = new InstanceVariableExample();
        obj1.displayValue();  // Output: 5

        InstanceVariableExample obj2 = new InstanceVariableExample();
        obj2.instanceVariable = 10;
        obj2.displayValue();  // Output: 10
    }
}

3. Class Variables (Static Variables)

Class variables are declared with the static keyword. These variables are shared among all instances of a class. They are accessible throughout the class and can be accessed using the class name or through instances.

Example:

public class ClassVariableExample {
    static int classVariable = 100;  // Static (class) variable

    public void displayValue() {
        System.out.println(classVariable);  // Accessible throughout the class
    }

    public static void main(String[] args) {
        ClassVariableExample obj1 = new ClassVariableExample();
        obj1.displayValue();  // Output: 100

        ClassVariableExample obj2 = new ClassVariableExample();
        obj2.classVariable = 200;
        obj2.displayValue();  // Output: 200
        System.out.println(ClassVariableExample.classVariable);  // Access via class name: Output: 200
    }
}

4. Parameters

Parameters are variables that are passed to methods or constructors. They exist only within the method or constructor they are passed to and are used to hold the data passed to the method.

Example:

public class ParameterExample {
    public void displaySum(int num1, int num2) {  // Parameters
        int sum = num1 + num2;
        System.out.println("Sum: " + sum);
    }

    public static void main(String[] args) {
        ParameterExample obj = new ParameterExample();
        obj.displaySum(5, 10);  // Output: Sum: 15
    }
}

5. Global Variables

In Java, there is no direct concept of global variables. However, instance and class variables can be used to mimic global variables when they are declared at the class level and accessed throughout the application.

Example:

public class GlobalVariableExample {
    static int globalVariable = 50;  // Simulated global variable

    public void displayValue() {
        System.out.println("Global Variable: " + globalVariable);  // Accessible throughout the class
    }

    public static void main(String[] args) {
        GlobalVariableExample obj1 = new GlobalVariableExample();
        obj1.displayValue();  // Output: Global Variable: 50

        GlobalVariableExample obj2 = new GlobalVariableExample();
        obj2.globalVariable = 100;
        obj2.displayValue();  // Output: Global Variable: 100
        System.out.println(GlobalVariableExample.globalVariable);  // Access via class name: Output: 100
    }
}
