Java Loops

Loops in Java

Loops are constructs that allow executing a block of code repeatedly, based on a condition. They are fundamental for iterating over data structures, processing arrays, performing repetitive tasks, or generating patterns.

Types of Loops in Java

Java provides the following types of loops:

  • Entry-controlled loops: Condition is checked before executing the loop body.
    • for loop
    • while loop
  • Exit-controlled loops: Loop body is executed at least once before the condition is checked.
    • do-while loop
  • Enhanced for loop: Specifically used for collections or arrays.

1. for Loop

Definition: The for loop is used when the number of iterations is known beforehand.

for (initialization; condition; update) {
    // Code to execute
}

Example: Counting Numbers

for (int i = 1; i <= 5; i++) {
    System.out.println("Iteration: " + i);
}

Explanation:

  • int i = 1: Start with i = 1.
  • i <= 5: Loop runs while i is less than or equal to 5.
  • i++: Increment i by 1 after each iteration.

Output:

Iteration: 1
Iteration: 2
Iteration: 3
Iteration: 4
Iteration: 5

2. while Loop

Definition: The while loop is used when the number of iterations is not fixed and depends on a condition.

while (condition) {
    // Code to execute
}

Example: Summing Numbers

int i = 1, sum = 0;
while (i <= 5) {
    sum += i;  // Add i to sum
    i++;       // Increment i
}
System.out.println("Sum: " + sum);

Explanation:

  • Start with i = 1, sum = 0.
  • As long as i <= 5, add i to sum and increment i.
  • Loop ends when i = 6.

Output:

Sum: 15

3. do-while Loop

Definition: The do-while loop ensures the loop body executes at least once, even if the condition is initially false.

do {
    // Code to execute
} while (condition);

Example: Displaying Numbers

int i = 6;
do {
    System.out.println("Value: " + i);
    i++;
} while (i <= 5);

Explanation:

  • Body executes once (Value: 6 is printed).
  • Condition i <= 5 evaluates to false, so the loop terminates.

Output:

Value: 6

4. Enhanced for Loop

Definition: An enhanced for loop (also called for-each loop) is used to iterate over arrays or collections.

for (type variable : array/collection) {
    // Code to execute
}

Example: Iterating Through an Array

int[] numbers = {1, 2, 3, 4, 5};

for (int num : numbers) {
    System.out.println("Number: " + num);
}

Explanation:

  • The loop iterates through each element in the array numbers.
  • Each element is assigned to num, which is then printed.

Output:

Number: 1
Number: 2
Number: 3
Number: 4
Number: 5

Entry-Controlled vs Exit-Controlled Loops

Criteria Entry-Controlled Loops Exit-Controlled Loops
Condition Check Condition is checked before executing the loop body. Condition is checked after executing the loop body.
Guarantee of Execution The loop may not execute if the condition is false initially. The loop will always execute at least once.
Common Types for, while loops. do-while loop.
Example for (int i = 0; i < 5; i++) do { // code } while (condition);
Use Case Used when the number of iterations is known beforehand. Used when the loop must run at least once, even if the condition is false.

Loop Control Statements

1. break Statement

Exits the loop immediately when encountered.

for (int i = 1; i <= 10; i++) {
    if (i == 5) break;
    System.out.println("Number: " + i);
}

Output:

Number: 1
Number: 2
Number: 3
Number: 4

2. continue Statement

Skips the current iteration and moves to the next.

for (int i = 1; i <= 5; i++) {
    if (i == 3) continue;
    System.out.println("Number: " + i);
}

Output:

Number: 1
Number: 2
Number: 4
Number: 5

Nested Loops

Loops can be nested to handle more complex tasks.

Example: Multiplication Table

for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        System.out.println(i + " * " + j + " = " + (i * j));
    }
}

Output:

1 * 1 = 1
1 * 2 = 2
1 * 3 = 3
2 * 1 = 2
...
3 * 3 = 9

Common Loop Mistakes

1. Infinite Loops

Forgetting to update loop variables:

while (true) {
    // Runs forever
}

2. Off-by-One Errors

Incorrect loop bounds cause under or over iteration.

Pattern Printing Example: Pyramid of Stars

public class Pyramid {
    public static void main(String[] args) {
        int rows = 5;

        for (int i = 1; i <= rows; i++) {
            for (int j = rows - i; j > 0; j--) {
                System.out.print(" ");
            }
            for (int k = 1; k <= (2 * i - 1); k++) {
                System.out.print("*");
            }
            System.out.println();
        }
    }
}

Output:

    *
   ***
  *****
 *******
*********

Key Points to Remember in Java Loops

  • Infinite Loops: Always make sure the loop condition will eventually evaluate to false to avoid infinite loops.
  • Loop Control Statements: The break statement terminates the loop, and continue skips the current iteration.
  • Off-by-One Errors: Be cautious about loop bounds and ensure you're iterating over the intended range.
  • Nested Loops: Be mindful of performance when using loops inside other loops, especially with large datasets.
  • Enhanced for Loop: Great for iterating through collections or arrays with simplicity and readability.
  • Condition Placement: Entry-controlled loops check conditions first, while exit-controlled loops check conditions after execution, so ensure your logic fits the type of loop you're using.
