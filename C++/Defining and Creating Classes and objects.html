Defining and Creating Classes and Objects in C++

Classes and Objects

C++ is an object-oriented programming (OOP) language, and understanding classes and objects is essential to mastering the language.
A class is a blueprint for creating objects, which are instances of the class. A class encapsulates data for the object and functions to manipulate that data.

Example of Class and Object

Class Structure

A class in C++ is a collection of data members (variables) and member functions (methods) grouped together. The data members represent the attributes of an object, and the member functions define the behavior or actions of that object.

Object Creation

An object is created by instantiating a class. Each object holds its own copy of the class’s data members but shares the same functions. Objects are the living instances of the blueprint defined by the class.

Defining a Class in C++

In C++, a class is defined using the class keyword, followed by the class name. The body of the class is enclosed within curly braces. You can specify access specifiers (private, public, protected) to control access to the members of the class.

Class Syntax

class ClassName {
    private:
        // Data members (variables)

    public:
        // Member functions (methods)
};

Example of Defining a Class

#include <iostream>
using namespace std;

class Car {
    private:
        string brand;  // Data member
        int year;      // Data member

    public:
        // Member function to set values
        void setCarInfo(string b, int y) {
            brand = b;
            year = y;
        }

        // Member function to display car information
        void displayInfo() {
            cout << "Car Brand: " << brand << endl;
            cout << "Car Year: " << year << endl;
        }
};

Explanation of the Class Definition

  • Data Members: The class has two private data members, brand and year. These variables hold information specific to each object of the class.
  • Member Functions: The class has two public member functions: setCarInfo (which sets the values for the car's brand and year) and displayInfo (which displays these values).
  • Access Specifiers: The members are divided into two sections: private (accessible only within the class) and public (accessible from outside the class).

Creating Objects in C++

Once a class is defined, objects can be created using the class as a blueprint. An object represents an instance of a class. When creating an object, memory is allocated for its data members, and the class’s member functions can be invoked using the object.

Object Creation Syntax

ClassName objectName;

Example of Object Creation

#include <iostream>
int main() {
    // Creating an object of the Car class
    Car myCar;

    // Setting values for the object using member functions
    myCar.setCarInfo("Toyota", 2022);

    // Displaying information using member function
    myCar.displayInfo();

    return 0;
}

Explanation of Object Creation

  • Creating an Object: We create an object of the class Car using the syntax Car myCar;.
  • Accessing Member Functions: The member function setCarInfo is called to initialize the object's attributes. Then, the displayInfo function is used to display those attributes.

Access Specifiers in C++

Access specifiers in C++ are used to control the visibility of class members. The three main access specifiers are:

  • private: Members are accessible only within the class and not from outside.
  • public: Members can be accessed from anywhere, both inside and outside the class.
  • protected: Members can be accessed within the class and by derived classes (in case of inheritance).

Constructors and Destructors in C++

In C++, classes can have special member functions known as constructors and destructors.

Constructor

A constructor is a special member function that initializes objects when they are created. It has the same name as the class and does not have a return type.

Destructor

A destructor is a special member function that is invoked when an object is destroyed. It is used to clean up resources that were allocated during the lifetime of the object.

Example with Constructor and Destructor

class Car {
    private:
        string brand;
        int year;

    public:
        // Constructor to initialize data members
        Car(string b, int y) {
            brand = b;
            year = y;
        }

        // Destructor
        ~Car() {
            cout << "The " << brand << " car is being destroyed." << endl;
        }

        // Function to display car details
        void displayInfo() {
            cout << "Car Brand: " << brand << endl;
            cout << "Car Year: " << year << endl;
        }
};

int main() {
    // Creating an object of Car, which calls the constructor
    Car myCar("Ford", 2021);
    myCar.displayInfo(); // Displaying car information
    
    return 0; // Destructor will be automatically called here
}

Explanation of Constructor and Destructor

  • Constructor: The constructor Car(string b, int y) is used to initialize the data members of the object when it is created. It is called automatically when the object myCar is created in main().
  • Destructor: The destructor ~Car() is invoked when the object is destroyed. It is called automatically when the program ends or the object goes out of scope. The destructor message is displayed as part of cleanup.
