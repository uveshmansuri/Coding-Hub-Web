Operators in C++

Introduction

In C++, operators are special symbols that perform operations on variables and values. These operators are essential in expressing computation, comparison, logical conditions, and more. C++ operators are divided into various categories based on their function and usage.

Types of Operators in C++

C++ supports a wide range of operators, which are categorized as:

  • Arithmetic Operators
  • Relational Operators
  • Logical Operators
  • Bitwise Operators
  • Assignment Operators
  • Increment/Decrement Operators
  • Conditional (Ternary) Operator
  • Sizeof Operator
  • Pointer Operators
  • Type-casting Operators

1. Arithmetic Operators

Arithmetic operators are used to perform basic mathematical operations. The following are the arithmetic operators in C++:

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • % : Modulus (remainder of division)
#include 
using namespace std;

int main() {
    int a = 10, b = 3;
    cout << "Addition: " << a + b << endl;     // 10 + 3 = 13
    cout << "Subtraction: " << a - b << endl;  // 10 - 3 = 7
    cout << "Multiplication: " << a * b << endl; // 10 * 3 = 30
    cout << "Division: " << a / b << endl;     // 10 / 3 = 3 (integer division)
    cout << "Modulus: " << a % b << endl;      // 10 % 3 = 1 (remainder)
    return 0;
}

Output:

Addition: 13
Subtraction: 7
Multiplication: 30
Division: 3
Modulus: 1

2. Relational Operators

Relational operators are used to compare two values. These operators return a boolean value: true (1) if the condition is satisfied, and false (0) otherwise. The relational operators in C++ are:

  • == : Equal to
  • != : Not equal to
  • > : Greater than
  • < : Less than
  • >= : Greater than or equal to
  • <= : Less than or equal to
#include 
using namespace std;

int main() {
    int a = 10, b = 5;
    cout << "a == b: " << (a == b) << endl; // False
    cout << "a != b: " << (a != b) << endl; // True
    cout << "a > b: " << (a > b) << endl;   // True
    cout << "a < b: " << (a < b) << endl;   // False
    cout << "a >= b: " << (a >= b) << endl; // True
    cout << "a <= b: " << (a <= b) << endl; // False
    return 0;
}

Output:

a == b: 0
a != b: 1
a > b: 1
a < b: 0
a >= b: 1
a <= b: 0

3. Logical Operators

Logical operators are used to combine conditional statements and return a boolean value. These are the logical operators in C++:

  • && : Logical AND
  • || : Logical OR
  • ! : Logical NOT
#include 
using namespace std;

int main() {
    bool a = true, b = false;
    cout << "(a && b): " << (a && b) << endl; // False (AND operation)
    cout << "(a || b): " << (a || b) << endl; // True (OR operation)
    cout << "!(a && b): " << !(a && b) << endl; // True (NOT operation)
    return 0;
}

Output:

(a && b): 0
(a || b): 1
!(a && b): 1

4. Bitwise Operators

Bitwise operators work at the bit level and perform operations on individual bits of integer values. The following are bitwise operators:

  • & : Bitwise AND
  • | : Bitwise OR
  • ^ : Bitwise XOR
  • ~ : Bitwise NOT
  • << : Left shift
  • > : Right shift
#include 
using namespace std;

int main() {
    int a = 5, b = 3;  // 5 = 0101, 3 = 0011 (binary)
    cout << "a & b: " << (a & b) << endl;  // AND -> 0001 (1 in decimal)
    cout << "a | b: " << (a | b) << endl;  // OR -> 0111 (7 in decimal)
    cout << "a ^ b: " << (a ^ b) << endl;  // XOR -> 0110 (6 in decimal)
    cout << "~a: " << (~a) << endl;        // NOT -> 1111...1010 (inverts bits)
    cout << "a << 1: " << (a << 1) << endl; // Left shift -> 1010 (10 in decimal)
    cout << "a >> 1: " << (a >> 1) << endl; // Right shift -> 0010 (2 in decimal)
    return 0;
}

Output:

a & b: 1
a | b: 7
a ^ b: 6
~a: -6
a << 1: 10
a >> 1: 2

5. Assignment Operators

Assignment operators are used to assign values to variables. Some of the commonly used assignment operators in C++ include:

  • = : Simple assignment
  • += : Add and assign
  • -= : Subtract and assign
  • *= : Multiply and assign
  • /= : Divide and assign
  • %= : Modulo and assign
#include 
using namespace std;

int main() {
    int a = 10;
    a += 5;  // a = a + 5
    cout << "a += 5: " << a << endl;  // 15
    a *= 2;  // a = a * 2
    cout << "a *= 2: " << a << endl;  // 30
    a /= 3;  // a = a / 3
    cout << "a /= 3: " << a << endl;  // 10
    return 0;
}

Output:

a += 5: 15
a *= 2: 30
a /= 3: 10
