Looping Statements in C++

Introduction to Loops

Loops are a fundamental concept in programming that allow us to repeat a block of code multiple times. This is useful in scenarios where you need to perform repetitive tasks without writing the same code over and over again. C++ provides several types of loops that give flexibility in controlling how many times a block of code should run.

There are three primary types of loops in C++:

  • for loop: Ideal when the number of iterations is known beforehand.
  • while loop: Suitable when the number of iterations is not known, and the loop continues as long as a condition is true.
  • do-while loop: Similar to the `while` loop but guarantees that the loop executes at least once.

The for Loop

The `for` loop is typically used when the number of iterations is known before the loop starts. It consists of three parts: initialization, condition, and increment/decrement. These parts are written inside parentheses and separated by semicolons. The loop runs as long as the condition is true.

Syntax:

for (initialization; condition; increment/decrement) {
    // Code to be executed
}

Explanation:

  • Initialization: This step is executed once before the loop starts. It is typically used to initialize the loop counter.
  • Condition: This condition is checked before every iteration. If the condition evaluates to true, the loop will execute. If false, the loop will terminate.
  • Increment/Decrement: After each iteration, the loop counter is updated based on the increment or decrement specified.

Example 1: Basic `for` loop

#include 
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        cout << "Iteration: " << i << endl;
    }
    return 0;
}

Output:

Iteration: 1
Iteration: 2
Iteration: 3
Iteration: 4
Iteration: 5

This example initializes the variable i to 1, and as long as i <= 5, the loop will print the current value of i and increment it by 1 after each iteration.

Example 2: Looping through an array using `for` loop

#include 
using namespace std;

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    for (int i = 0; i < 5; i++) {
        cout << "Element " << i+1 << ": " << arr[i] << endl;
    }
    return 0;
}

Output:

Element 1: 1
Element 2: 2
Element 3: 3
Element 4: 4
Element 5: 5

The while Loop

The `while` loop is used when the number of iterations is unknown. The loop continues to execute as long as the condition evaluates to true. The condition is evaluated before the execution of the loop body.

Syntax:

while (condition) {
    // Code to be executed
}

Example 1: Basic `while` loop

#include 
using namespace std;

int main() {
    int i = 1;
    while (i <= 5) {
        cout << "Iteration: " << i << endl;
        i++;
    }
    return 0;
}

Output:

Iteration: 1
Iteration: 2
Iteration: 3
Iteration: 4
Iteration: 5

In this example, the loop runs while the condition i <= 5 holds true. The counter i is incremented inside the loop to eventually break the loop.

Example 2: Using `while` loop with user input

#include 
using namespace std;

int main() {
    int num;
    cout << "Enter a number (0 to exit): ";
    cin >> num;
    
    while (num != 0) {
        cout << "You entered: " << num << endl;
        cout << "Enter a number (0 to exit): ";
        cin >> num;
    }

    cout << "Exiting program." << endl;
    return 0;
}

Output:

Enter a number (0 to exit): 5
You entered: 5
Enter a number (0 to exit): 10
You entered: 10
Enter a number (0 to exit): 0
Exiting program.

The do-while Loop

The `do-while` loop is similar to the `while` loop, except that the condition is checked after the loop's body is executed. This ensures that the loop executes at least once, even if the condition is initially false.

Syntax:

do {
    // Code to be executed
} while (condition);

Example 1: Basic `do-while` loop

#include 
using namespace std;

int main() {
    int i = 1;
    do {
        cout << "Iteration: " << i << endl;
        i++;
    } while (i <= 5);
    return 0;
}

Output:

Iteration: 1
Iteration: 2
Iteration: 3
Iteration: 4
Iteration: 5

Here, the loop executes at least once, even if the condition is initially false, but in this case, the condition i <= 5 ensures that it runs until i exceeds 5.

Example 2: User input with `do-while` loop

#include 
using namespace std;

int main() {
    int num;
    do {
        cout << "Enter a number (0 to exit): ";
        cin >> num;
        cout << "You entered: " << num << endl;
    } while (num != 0);

    cout << "Exiting program." << endl;
    return 0;
}

Output:

Enter a number (0 to exit): 5
You entered: 5
Enter a number (0 to exit): 10
You entered: 10
Enter a number (0 to exit): 0
You entered: 0
Exiting program.

Break and Continue

In C++, the `break` and `continue` statements provide additional control over loops:

  • break: Exits the loop prematurely.
  • continue: Skips the current iteration and proceeds with the next one.

Example: Using `break` to exit the loop

#include 
using namespace std;

int main() {
    for (int i = 1; i <= 10; i++) {
        if (i == 5) {
            break; // Exit the loop when i is 5
        }
        cout << "Iteration: " << i << endl;
    }
    return 0;
}

Output:

Iteration: 1
Iteration: 2
Iteration: 3
Iteration: 4

Example: Using `continue` to skip an iteration

#include 
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            continue; // Skip iteration 3
        }
        cout << "Iteration: " << i << endl;
    }
    return 0;
}

Output:

Iteration: 1
Iteration: 2
Iteration: 4
Iteration: 5
