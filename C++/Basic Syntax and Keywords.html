Basic Syntax and Keywords in C++

Introduction

C++ is a powerful, high-performance programming language used for system software, game development, and applications that require real-time processing. It builds upon the C programming language and adds object-oriented features, such as classes and inheritance, among other improvements. Understanding the basic syntax and keywords in C++ is essential for writing correct and efficient programs.

Basic Syntax in C++

C++ programs are written using a specific structure, and understanding this syntax is crucial for building effective programs. Below are the key elements of C++ syntax:

1. Structure of a C++ Program

A simple C++ program has the following structure:

#include <iostream>

using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

Let’s break down the components of this program:

  • #include <iostream>: This is a preprocessor directive that tells the compiler to include the iostream library, which is necessary for input and output operations.
  • using namespace std; This allows us to use standard library objects and functions (like cout) without the std:: prefix.
  • int main(): This is the main function where execution starts. It must return an integer (typically 0 to indicate successful completion).
  • cout << "Hello, World!" << endl; This is a statement that outputs the string "Hello, World!" to the console.
  • return 0; This returns an integer value to the operating system, indicating the program has executed successfully.

2. Comments in C++

Comments are essential for making your code understandable to others (and to yourself). C++ supports two types of comments:

  • Single-line comment: These comments are written on a single line using // before the comment text.
    • // This is a single-line comment
  • Multi-line comment: These comments span multiple lines and are enclosed between /* and */.
    • /*
            This is a
            multi-line comment
            */

3. Data Types

C++ provides a variety of built-in data types, such as:

  • int: Integer data type used for whole numbers.
  • float: Used for single-precision floating-point numbers (decimals).
  • double: Used for double-precision floating-point numbers.
  • char: Used for single characters (e.g., 'a', '1').
  • bool: Used for boolean values (true or false).
  • string: Represents a sequence of characters (typically used with the string class).

4. Variables and Constants

Variables are used to store data values. In C++, variables must be declared before they are used. For example:

int age = 25;
float pi = 3.14;
string name = "John";

Constants, on the other hand, are values that do not change during the program’s execution. Constants are declared using the const keyword:

const int DAYS_IN_WEEK = 7;

5. Operators

Operators are symbols used to perform operations on variables and values. There are various types of operators in C++:

  • Arithmetic Operators: + (addition), - (subtraction), * (multiplication), / (division), % (modulus).
  • Relational Operators: == (equal to), != (not equal to), > (greater than), < (less than), >= (greater than or equal to), <= (less than or equal to).
  • Logical Operators: && (logical AND), || (logical OR), ! (logical NOT).
  • Assignment Operators: = (assign), += (add and assign), -= (subtract and assign), etc.

6. Control Structures

C++ uses control structures such as if, else, for, while, and switch to control the flow of execution in a program.

if statement:

if (age > 18) {
    cout << "You are an adult." << endl;
} else {
    cout << "You are a minor." << endl;
}

for loop:

for (int i = 0; i < 5; i++) {
    cout << "i = " << i << endl;
}

while loop:

int i = 0;
while (i < 5) {
    cout << "i = " << i << endl;
    i++;
}

switch statement:

switch (day) {
    case 1:
        cout << "Monday" << endl;
        break;
    case 2:
        cout << "Tuesday" << endl;
        break;
    default:
        cout << "Invalid day" << endl;
}

Keywords in C++

Keywords are reserved words in C++ that have predefined meanings. These cannot be used as identifiers (names for variables, functions, etc.). Below is a list of some of the most commonly used keywords in C++:

1. Control Flow Keywords

  • if, else: Used to create conditional statements.
  • for, while, do: Used for looping.
  • switch, case, default: Used to implement switch-case statements.

2. Data Type Keywords

  • int: Defines integer type variables.
  • float, double: Defines floating-point number types.
  • char: Defines character type variables.
  • bool: Defines boolean type variables (true or false).

3. Other Important Keywords

  • return: Used to exit a function and return a value.
  • void: Specifies a function that does not return a value.
  • const: Defines a constant value.
  • new, delete: Used for dynamic memory allocation and deallocation.
