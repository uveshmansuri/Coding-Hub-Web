C++ Strings and String Functions

1. C++ Strings Overview

C++ strings are used to represent a sequence of characters. There are two main types of strings in C++:

  • C-style Strings (Character Arrays): These are simple arrays of characters ending with a null character (`'\0'`).
  • std::string (C++ String Class): This is a dynamic string representation provided by the C++ Standard Library, which is more flexible and easier to work with.

1.1 C-style Strings

C-style strings are arrays of characters, terminated with a null character `'\0'`. You have to manually manage memory when using them, and operations like string length or copying require specific functions from the `` library.


#include 
using namespace std;

int main() {
    // C-style string (character array)
    char str[] = "Hello, World!";
    
    // Printing the C-style string
    cout << "C-style string: " << str << endl;

    return 0;
}

Output:


C-style string: Hello, World!

1.2 C++ std::string

The std::string class is a much more flexible and powerful way to work with strings in C++. It automatically handles memory allocation and provides many built-in methods to manipulate the string.


#include 
#include 
using namespace std;

int main() {
    // C++ string
    string str = "Hello, World!";
    
    // Printing the C++ string
    cout << "C++ string: " << str << endl;

    return 0;
}

Output:


C++ string: Hello, World!

2. String Functions in C++

C++ provides a rich set of functions to perform various operations on strings. The most important functions are included in the std::string class. These functions provide capabilities such as string length, concatenation, comparison, searching, and more.

2.1 Length of String

The size() or length() function returns the number of characters in a string, excluding the null terminator. Both functions work identically and can be used interchangeably.


#include 
#include 
using namespace std;

int main() {
    string str = "Hello, World!";
    
    // Getting the length of the string
    cout << "Length of string: " << str.size() << endl;

    return 0;
}

Output:


Length of string: 13

2.2 String Concatenation

The + operator is used to concatenate two or more strings in C++.


#include 
#include 
using namespace std;

int main() {
    string str1 = "Hello, ";
    string str2 = "World!";
    
    // Concatenating strings
    string result = str1 + str2;

    cout << "Concatenated string: " << result << endl;

    return 0;
}

Output:


Concatenated string: Hello, World!

2.3 Accessing Characters in a String

You can access individual characters in a string using the array subscript operator ([]) or the at() method. The at() method provides bounds checking and throws an exception if the index is out of range.


#include 
#include 
using namespace std;

int main() {
    string str = "Hello, World!";
    
    // Accessing characters using []
    cout << "First character: " << str[0] << endl;
    cout << "Last character: " << str.at(str.size() - 1) << endl;

    return 0;
}

Output:


First character: H
Last character: !

2.4 String Comparison

C++ allows you to compare strings using the ==, !=, <, >, and compare() methods. You can also use the compare() method for case-sensitive comparisons.


#include 
#include 
using namespace std;

int main() {
    string str1 = "Hello";
    string str2 = "World";
    
    if (str1 == str2) {
        cout << "Strings are equal." << endl;
    } else {
        cout << "Strings are not equal." << endl;
    }

    return 0;
}

Output:


Strings are not equal.

2.5 Substring

The substr() function extracts a substring from a string. It takes two parameters: the starting index and the length of the substring.


#include 
#include 
using namespace std;

int main() {
    string str = "Hello, World!";
    
    // Extracting a substring
    string sub = str.substr(7, 5); // Starts from index 7, length 5
    cout << "Substring: " << sub << endl;

    return 0;
}

Output:


Substring: World

2.6 String Search

You can search for a substring within a string using the find() method. If the substring is found, it returns the index of the first occurrence, otherwise it returns string::npos.


#include 
#include 
using namespace std;

int main() {
    string str = "Hello, World!";
    
    // Finding a substring
    size_t found = str.find("World");
    if (found != string::npos) {
        cout << "'World' found at index: " << found << endl;
    } else {
        cout << "'World' not found!" << endl;
    }

    return 0;
}

Output:


'World' found at index: 7

3. Other Useful String Functions

Here are some additional useful string functions in C++:

  • empty(): Checks if the string is empty.
  • erase(): Removes characters from a string.
  • insert(): Inserts a substring at a specified position.
  • replace(): Replaces a substring with another substring.
