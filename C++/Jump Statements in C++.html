Jump Statements in C++

Introduction to Jump Statements

Jump statements in C++ provide control over the flow of execution within a program. These statements are powerful tools used to alter the normal sequence of operations in loops, conditionals, and even functions. In C++, there are three main jump statements:

  • break: Exits from a loop or switch statement prematurely.
  • continue: Skips the current iteration of a loop and proceeds with the next iteration.
  • goto: Jumps to a specified label in the program, transferring control to that location.

While these statements are useful in some scenarios, they should be used carefully to avoid making the code harder to read and maintain.

The break Statement

The `break` statement is used to exit a loop (like `for`, `while`, or `do-while`) or a `switch` case statement before it completes normally. This allows you to terminate the loop or switch early based on some condition.

Syntax:

break;

Example 1: Using break in a for loop


#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 10; i++) {
        if (i == 5) {
            break; // Exit the loop when i is 5
        }
        cout << "Iteration: " << i << endl;
    }
    return 0;
}

Output:


Iteration: 1
Iteration: 2
Iteration: 3
Iteration: 4

In this example, the loop stops executing when the value of i reaches 5 because the break statement is encountered.

Example 2: Using break in a switch statement


#include <iostream>
using namespace std;

int main() {
    int number = 2;
    switch (number) {
        case 1:
            cout << "One" << endl;
            break;
        case 2:
            cout << "Two" << endl;
            break;
        case 3:
            cout << "Three" << endl;
            break;
        default:
            cout << "Default" << endl;
    }
    return 0;
}

Output:


Two

In the `switch` statement example, the `break` ensures that the program exits the switch statement after the matching case has executed. Without it, the program would "fall through" to the next case.

The continue Statement

The `continue` statement is used to skip the current iteration of a loop and continue with the next iteration. It is often used when a specific condition occurs, and we want to skip the rest of the code inside the loop for that particular iteration.

Syntax:

continue;

Example 1: Using continue in a for loop


#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            continue; // Skip iteration when i is 3
        }
        cout << "Iteration: " << i << endl;
    }
    return 0;
}

Output:


Iteration: 1
Iteration: 2
Iteration: 4
Iteration: 5

The iteration for i = 3 is skipped due to the continue statement. The loop continues for the remaining values of i.

Example 2: Using continue in a while loop


#include <iostream>
using namespace std;

int main() {
    int i = 0;
    while (i < 5) {
        i++;
        if (i == 3) {
            continue; // Skip iteration when i is 3
        }
        cout << "Iteration: " << i << endl;
    }
    return 0;
}

Output:


Iteration: 1
Iteration: 2
Iteration: 4
Iteration: 5

The continue statement skips printing when i equals 3 in the loop.

The goto Statement

The `goto` statement is a way to jump to a specific part of the program. This is done using a label, which is a user-defined identifier followed by a colon. Once the `goto` statement is encountered, the control jumps to the label location in the code.

Syntax:


label_name:
    // Code to be executed
goto label_name;

Example 1: Using goto to jump to a label


#include <iostream>
using namespace std;

int main() {
    int i = 1;
    if (i == 1) {
        goto skip; // Jump to skip label
    }
    cout << "This will be skipped." << endl;

skip:
    cout << "This will be printed." << endl;
    return 0;
}

Output:


This will be printed.

In this example, the `goto` statement skips the `cout` statement before the label `skip` and jumps directly to the code after it.

Example 2: Using goto in a nested loop


#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 3; i++) {
        for (int j = 1; j <= 3; j++) {
            if (i == 2 && j == 2) {
                goto exitLoop; // Exit both loops when i is 2 and j is 2
            }
            cout << "i: " << i << " j: " << j << endl;
        }
    }
    
exitLoop:
    cout << "Exited the loops." << endl;
    return 0;
}

Output:


i: 1 j: 1
i: 1 j: 2
i: 1 j: 3
i: 2 j: 1
Exited the loops.

In this example, the `goto` statement is used to exit both loops when i = 2 and j = 2.
