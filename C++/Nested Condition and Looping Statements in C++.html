Nested Conditionals and Looping Statements in C++

Introduction

In C++, both conditional statements and loops are fundamental building blocks of any program. However, these control structures can be nested to handle more complex problems. Nesting allows us to put one conditional or looping structure inside another, giving us more flexibility and power in program logic. In this section, we will look at how nested conditionals and loops work and how they can be used together to solve complex problems.

While nested structures are useful, they should be used with care. Excessive nesting can lead to less readable code, so it is important to use these structures judiciously.

1. Nested Conditional Statements

Conditional statements control the flow of execution based on conditions. When one conditional statement is placed inside another, it is called a "nested conditional." This is useful when we need to make multiple, dependent decisions based on a set of conditions.

Nested `if` Statements

Nested `if` statements occur when you have an `if` statement inside another `if` block. This is often used when checking multiple conditions that depend on each other.

Syntax


if (condition1) {
    if (condition2) {
        // Code block executed when both conditions are true
    }
}

Example: Checking Age and License


#include <iostream>
using namespace std;

int main() {
    int age = 20;
    bool hasLicense = false;

    if (age >= 18) {
        if (hasLicense) {
            cout << "You are eligible to drive!" << endl;
        } else {
            cout << "You need a driving license to drive!" << endl;
        }
    } else {
        cout << "You are not old enough to drive!" << endl;
    }
    return 0;
}

Output:


You need a driving license to drive!

In this example, the program first checks whether the person is old enough to drive. If true, it then checks whether the person has a license. This is an example of a nested `if` statement, where the second `if` only runs if the first `if` is true.

Nested `else if` Statements

We can also use `else if` to create a chain of conditional checks. A nested `else if` allows for multiple possible outcomes based on different conditions.

Example: Grading System


#include <iostream>
using namespace std;

int main() {
    int score = 85;

    if (score >= 90) {
        cout << "Grade A" << endl;
    } else if (score >= 75) {
        cout << "Grade B" << endl;
    } else if (score >= 60) {
        cout << "Grade C" << endl;
    } else {
        cout << "Grade F" << endl;
    }
    return 0;
}

Output:


Grade B

This example shows how multiple conditions are checked one by one. If the first condition is false, the program proceeds to the next `else if`, and so on.

2. Nested Looping Statements

In C++, loops can be nested within one another, which is known as a "nested loop." A nested loop allows you to perform iterations within iterations, such as printing out a table of values or traversing multi-dimensional arrays.

Nested `for` Loops

A nested `for` loop is commonly used when working with multi-dimensional data or needing to perform multiple iterations of a set of actions.

Syntax


for (int i = 0; i < n; i++) {
    for (int j = 0; j < m; j++) {
        // Code to be executed
    }
}

Example: Multiplication Table


#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 3; i++) {
        for (int j = 1; j <= 3; j++) {
            cout << i * j << "\t"; // Print multiplication result
        }
        cout << endl; // Move to next line after inner loop
    }
    return 0;
}

Output:


1	2	3	
2	4	6	
3	6	9

This code demonstrates how nested `for` loops are used to create a multiplication table. The outer loop controls the row, and the inner loop controls the column.

Nested `while` Loops

You can also nest `while` loops, where the outer loop condition controls the number of iterations of the inner `while` loop.

Example: Counting Numbers in a Grid


#include <iostream>
using namespace std;

int main() {
    int i = 1;
    while (i <= 3) {
        int j = 1;
        while (j <= 3) {
            cout << "i = " << i << ", j = " << j << endl;
            j++;
        }
        i++;
    }
    return 0;
}

Output:


i = 1, j = 1
i = 1, j = 2
i = 1, j = 3
i = 2, j = 1
i = 2, j = 2
i = 2, j = 3
i = 3, j = 1
i = 3, j = 2
i = 3, j = 3

This example uses nested `while` loops to print the combination of values from two variables `i` and `j` within the specified ranges.
