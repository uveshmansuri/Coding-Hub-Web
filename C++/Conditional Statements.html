Conditional Statements in C++

Introduction

Conditional statements in C++ are used to execute a block of code based on whether a specific condition is true or false. They provide the ability to make decisions within your program. The decision-making statements in C++ are:

  • if statement - Executes a block of code if the condition is true.
  • if-else statement - Executes one block of code if the condition is true, and another block if it is false.
  • else-if ladder - Used for multiple conditions, where each condition is checked in sequence.
  • switch-case statement - An alternative to the if-else chain, useful for selecting one out of many possible cases.

1. if Statement

The if statement checks a condition (expression). If the condition is true, the block of code inside the if statement is executed.

Example:

#include 
using namespace std;

int main() {
    int num = 10;
    if (num > 0) {  // Checks if num is greater than zero
        cout << "The number is positive." << endl; // Executed if condition is true
    }
    return 0;
}

In this example, the condition num > 0 evaluates to true, so the output will be:

The number is positive.

2. if-else Statement

The if-else statement is an extension of the if statement. It executes one block of code if the condition is true and another block of code if the condition is false.

Example:

#include 
using namespace std;

int main() {
    int num = -5;
    if (num > 0) {
        cout << "The number is positive." << endl;
    } else {
        cout << "The number is negative." << endl;
    }
    return 0;
}

Here, the condition num > 0 is false, so the program executes the else block, outputting:

The number is negative.

3. else-if Ladder

The else-if ladder allows you to check multiple conditions. If one condition is true, it executes that block of code and skips the others. It’s useful when there are several possible conditions to check.

Example:

#include 
using namespace std;

int main() {
    int num = 0;
    if (num > 0) {
        cout << "The number is positive." << endl;
    } else if (num < 0) {
        cout << "The number is negative." << endl;
    } else {
        cout << "The number is zero." << endl;
    }
    return 0;
}

In this example, since num == 0, the condition num == 0 is true, so the output will be:

The number is zero.

4. switch-case Statement

The switch-case statement is an alternative to the if-else ladder, especially when dealing with multiple values of a single variable. It allows you to test a variable against a list of constants and execute the corresponding block of code for the first match. If no case matches, the default block is executed.

Example:

#include 
using namespace std;

int main() {
    int day = 3;  // day variable
    switch (day) {
        case 1:
            cout << "Monday" << endl;
            break;
        case 2:
            cout << "Tuesday" << endl;
            break;
        case 3:
            cout << "Wednesday" << endl; // case 3
            break;
        case 4:
            cout << "Thursday" << endl;
            break;
        case 5:
            cout << "Friday" << endl;
            break;
        case 6:
            cout << "Saturday" << endl;
            break;
        case 7:
            cout << "Sunday" << endl;
            break;
        default:
            cout << "Invalid day" << endl; // if no case matches
    }
    return 0;
}

Here, the value of day is 3, so the program prints "Wednesday". If the value of day was not in the range 1-7, it would print "Invalid day".

Output:

Wednesday

Understanding the Flow of Conditional Statements

Conditional statements are key to controlling the flow of a program. Depending on the truth value of the expression in the conditional statement, the program will decide which set of instructions to follow.

For example:

  • If you want to check if a user is eligible for a particular offer, you might use an if statement to check if their age is above a certain value.
  • For deciding between multiple choices, you might use an else-if ladder to evaluate each condition in sequence.
  • If you have many possible cases (e.g., days of the week or different types of fruits), the switch-case structure is ideal for readability and efficiency.

Conclusion

Conditional statements are one of the fundamental building blocks in C++ programming, enabling you to create programs that can make decisions and react to different situations. By mastering if, else, else-if, and switch-case, you can control the flow of your program to handle different scenarios efficiently and effectively.
