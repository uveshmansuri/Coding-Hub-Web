C++ Functions

Introduction to Functions

A function in C++ is a block of code that performs a specific task. Functions allow you to break your program into smaller, manageable parts, making it easier to write, read, and maintain. Functions can take inputs (parameters), perform operations, and return results.

Syntax of a Function

The general syntax of a function in C++ is as follows:


return_type function_name(parameter_list) {
    // function body
}

Where:

  • return_type: The type of value the function will return (e.g., int, float, void for no return value).
  • function_name: The name of the function.
  • parameter_list: A list of input parameters (optional, can be empty).

Example of a Simple Function

This is an example of a function that adds two integers:


#include 
using namespace std;

// Function declaration
int add(int a, int b);

int main() {
    int result = add(10, 20);  // Function call
    cout << "The sum is: " << result << endl;
    return 0;
}

// Function definition
int add(int a, int b) {
    return a + b;
}

Output:


The sum is: 30

Types of Functions

C++ functions can be classified into the following types:

1. Standard Functions

These are functions that come with C++ libraries, such as main(), printf(), etc.

2. User-Defined Functions

Functions created by the user to perform specific tasks are called user-defined functions. These can be further classified based on their behavior:

  • Functions with arguments and return values
  • Functions with arguments but no return value (void functions)
  • Functions without arguments and return values
  • Functions without arguments and no return value (void functions)

3. Void Functions

A function with a return type of void does not return any value. These types of functions are useful when we need to perform an action but don’t require a result.


#include 
using namespace std;

void greet() {
    cout << "Hello, World!" << endl;
}

int main() {
    greet(); // Function call
    return 0;
}

Output:


Hello, World!

Function Overloading

C++ allows multiple functions with the same name but different parameter types or number of parameters. This is called function overloading. The correct function is chosen based on the number and types of arguments passed during the function call.

Example of Function Overloading


#include 
using namespace std;

int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}

int main() {
    cout << "Sum of integers: " << add(10, 20) << endl;
    cout << "Sum of doubles: " << add(10.5, 20.5) << endl;
    return 0;
}

Output:


Sum of integers: 30
Sum of doubles: 31

Function Recursion

A function that calls itself is called a recursive function. Recursion is a powerful technique that can simplify certain types of problems, such as computing factorials or solving the Fibonacci sequence.

Example of Recursion

Here’s an example of a recursive function to calculate the factorial of a number:


#include 
using namespace std;

int factorial(int n) {
    if (n == 0) {
        return 1;  // Base case
    } else {
        return n * factorial(n - 1); // Recursive call
    }
}

int main() {
    int result = factorial(5);
    cout << "Factorial of 5 is: " << result << endl;
    return 0;
}

Output:


Factorial of 5 is: 120

Function Parameters

Functions can accept parameters, which are values passed into the function when it is called. C++ supports different types of parameters:

Pass by Value

In pass-by-value, the actual parameter is copied into the formal parameter. Changes made to the parameter do not affect the actual argument.


#include 
using namespace std;

void increment(int x) {
    x = x + 1;
}

int main() {
    int num = 10;
    increment(num);  // Pass by value
    cout << "Value of num: " << num << endl; // Original value of num remains unchanged
    return 0;
}

Output:


Value of num: 10

Pass by Reference

In pass-by-reference, the function works directly with the memory address of the argument, so changes made to the parameter will affect the actual argument.


#include 
using namespace std;

void increment(int &x) {
    x = x + 1;
}

int main() {
    int num = 10;
    increment(num);  // Pass by reference
    cout << "Value of num: " << num << endl; // num is now modified
    return 0;
}

Output:


Value of num: 11
