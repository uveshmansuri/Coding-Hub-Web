Input and Output Functions in C++

Introduction

C++ provides essential functions for input and output operations. The input is generally read from the user, and the output is typically displayed on the console or stored in a file. The primary functions for input and output in C++ are cin for reading input and cout for displaying output. Both of these functions are part of the iostream header file.

1. Output in C++ with cout

The cout function is used to output data to the console in C++. It is a standard output stream, and you can display text, variables, and other values. You use the << operator to send data to the output stream.

Basic Syntax:

#include 
using namespace std;

int main() {
    cout << "Hello, World!";  // Simple output statement
    return 0;
}

This is a simple C++ program where cout displays the message "Hello, World!" on the screen.

Displaying Different Data Types:

#include 
using namespace std;

int main() {
    int age = 25;
    double pi = 3.14159;
    char grade = 'A';
    bool isStudent = true;

    cout << "Age: " << age << endl;
    cout << "Pi: " << pi << endl;
    cout << "Grade: " << grade << endl;
    cout << "Is student: " << isStudent << endl;

    return 0;
}

In the example, cout is used to output variables of different types, such as int, double, char, and bool. The endl is used to insert a newline after each output to move to the next line.

Multiple Outputs in One Line:

#include 
using namespace std;

int main() {
    int age = 30;
    double height = 5.9;

    cout << "Age: " << age << ", Height: " << height << endl;
    return 0;
}

You can also output multiple values in a single line by using multiple << operators. In this case, the age and height are displayed in the same line, separated by a comma.

2. Input in C++ with cin

The cin function is used to get input from the user. It reads data from the standard input (usually the keyboard) and stores it in a variable. The >> operator is used to read data from the input stream.

Basic Syntax:

#include 
using namespace std;

int main() {
    int age;
    cout << "Enter your age: ";
    cin >> age;  // Input a value for the variable 'age'
    cout << "You entered: " << age << endl;

    return 0;
}

This program prompts the user to enter their age, which is stored in the age variable. Then, the program outputs the entered age.

Reading Multiple Inputs:

#include 
using namespace std;

int main() {
    int age;
    double height;
    char grade;
    
    cout << "Enter your age: ";
    cin >> age;

    cout << "Enter your height (in meters): ";
    cin >> height;

    cout << "Enter your grade: ";
    cin >> grade;

    cout << "Age: " << age << ", Height: " << height << ", Grade: " << grade << endl;

    return 0;
}

This example demonstrates how to input different types of data: an integer, a floating-point number, and a character. These are stored in the variables age, height, and grade, respectively.

3. Error Handling in Input

When using cin, if the user enters incorrect data (e.g., entering text when an integer is expected), it can cause the program to behave unexpectedly. To handle input errors, we use functions like cin.fail(), cin.clear(), and cin.ignore() to recover from errors.

Handling Invalid Input:

#include 
using namespace std;

int main() {
    int age;
    cout << "Enter your age: ";
    cin >> age;

    if (cin.fail()) {  // Check for invalid input
        cout << "Invalid input! Please enter a valid integer." << endl;
        cin.clear();  // Clear error flag
        cin.ignore();  // Ignore incorrect input in the buffer
    } else {
        cout << "You entered: " << age << endl;
    }

    return 0;
}

In the above example, if the user enters a non-integer, the program will output an error message and attempt to clear the input buffer using cin.clear() and cin.ignore().

4. Formatting Output in C++

C++ provides several output formatting tools. You can set the width of the output, control precision for floating-point numbers, and use manipulators like setw, fixed, and setprecision to format your output.

Setting Output Width and Precision:

#include 
#include   // For output formatting manipulators
using namespace std;

int main() {
    double pi = 3.14159265358979;

    cout << "Pi with 2 decimal places: " << fixed << setprecision(2) << pi << endl;
    cout << "Pi with 4 decimal places: " << fixed << setprecision(4) << pi << endl;

    return 0;
}

In the example, the fixed manipulator ensures that the floating-point number is displayed in fixed-point notation. The setprecision manipulator controls the number of digits displayed after the decimal point.

Aligning Output:

#include 
#include 
using namespace std;

int main() {
    cout << setw(10) << "Hello" << endl;  // Set width to 10
    cout << setw(10) << "World" << endl;  // Right align

    return 0;
}

The setw manipulator is used to set the width of the output. This is useful when you need to format tables or columns of data neatly aligned.
