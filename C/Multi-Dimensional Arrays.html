Multi-Dimensional Arrays in C

What is a Multi-Dimensional Array?

A multi-dimensional array in C is an array of arrays. It allows you to store data in a tabular form, such as rows and columns. The most commonly used multi-dimensional array is a two-dimensional array, but arrays with more than two dimensions are also supported.

Multi-dimensional arrays are especially useful for representing complex data structures such as matrices, tables, and grids.

Declaration of Multi-Dimensional Arrays

The general syntax for declaring a multi-dimensional array is:

data_type array_name[size1][size2]...[sizeN];

For example, to declare a 2D array of integers with 3 rows and 4 columns:

int matrix[3][4];

To declare a 3D array:

int tensor[2][3][4];

Initialization of Multi-Dimensional Arrays

Multi-dimensional arrays can be initialized at the time of declaration. For example:

int matrix[2][3] = {
    {1, 2, 3},  // First row
    {4, 5, 6}   // Second row
};

You can also omit the inner curly braces, and the compiler will infer them:

int matrix[2][3] = {1, 2, 3, 4, 5, 6};

If you don't specify all the values, the remaining elements will be initialized to 0:

int matrix[2][3] = {{1, 2}, {4}};  // Remaining elements will be 0

Accessing Elements of a Multi-Dimensional Array

To access an element in a multi-dimensional array, specify the indices of each dimension in square brackets. For example:

int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

printf("%d", matrix[1][2]);  // Access element at row 1, column 2 (Output: 6)

Example: Two-Dimensional Array

The following example demonstrates the use of a 2D array:

#include <stdio.h>

int main() {
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    // Display the array elements
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            printf("matrix[%d][%d] = %d\n", i, j, matrix[i][j]);
        }
    }

    return 0;
}

Output:

matrix[0][0] = 1
matrix[0][1] = 2
matrix[0][2] = 3
matrix[1][0] = 4
matrix[1][1] = 5
matrix[1][2] = 6

Example: Matrix Addition

The following example demonstrates the addition of two 2D arrays:

#include <stdio.h>

int main() {
    int matrix1[2][2] = {{1, 2}, {3, 4}};
    int matrix2[2][2] = {{5, 6}, {7, 8}};
    int result[2][2];

    // Add the matrices
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 2; j++) {
            result[i][j] = matrix1[i][j] + matrix2[i][j];
        }
    }

    // Display the result
    printf("Resultant Matrix:\n");
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 2; j++) {
            printf("%d ", result[i][j]);
        }
        printf("\n");
    }

    return 0;
}

Output:

Resultant Matrix:
6 8
10 12

Higher-Dimensional Arrays

While 2D arrays are the most common, you can define arrays with three or more dimensions. For example, a 3D array:

int tensor[2][2][3] = {
    {
        {1, 2, 3}, {4, 5, 6}  // First layer
    },
    {
        {7, 8, 9}, {10, 11, 12}  // Second layer
    }
};

// Accessing an element
printf("%d", tensor[1][0][2]);  // Output: 9

Applications of Multi-Dimensional Arrays

Multi-dimensional arrays are widely used in various applications, such as:

  • Representing matrices in mathematical computations.
  • Storing tabular data, such as student marks or employee records.
  • Developing games that require grids (e.g., chess boards).
  • Handling multi-channel images in image processing.

Conclusion

Multi-dimensional arrays in C provide a powerful way to store and manipulate structured data. Understanding how to declare, initialize, and use them is essential for solving complex programming problems.
