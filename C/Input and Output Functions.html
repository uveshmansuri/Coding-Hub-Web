Input and Output Functions in C

Introduction

The input and output functions are essential for interacting with users in a C program. These functions enable data input from the user and output of results. In C, the standard functions scanf and printf are used for these purposes.

The printf Function

The printf function is used to display information on the screen. It is part of the standard input-output library (stdio.h).

Syntax:

printf("format string", argument1, argument2, ...);

Format Specifiers:

Specifier Data Type Example
%d Integer printf("Number: %d", 10);
%f Floating-point printf("Value: %f", 3.14);
%c Character printf("Grade: %c", 'A');
%s String printf("Name: %s", "John");

Example:

#include <stdio.h>

int main() {
    int num = 10;
    float pi = 3.14;
    char grade = 'A';

    printf("Number: %d\\n", num);
    printf("Value of Pi: %.2f\\n", pi);
    printf("Grade: %c\\n", grade);

    return 0;
}

Output:

  • Number: 10
  • Value of Pi: 3.14
  • Grade: A

The scanf Function

The scanf function is used to read user input. It reads data from standard input (keyboard) and assigns it to the specified variables.

Syntax:

scanf("format string", &variable1, &variable2, ...);

Note: The & (address-of operator) is used to provide the address of the variable where the input value will be stored.

Example:

#include <stdio.h>

int main() {
    int age;
    float height;
    char name[20];

    printf("Enter your age: ");
    scanf("%d", &age);

    printf("Enter your height: ");
    scanf("%f", &height);

    printf("Enter your name: ");
    scanf("%s", name);

    printf("\\nYour Details:\\n");
    printf("Name: %s\\n", name);
    printf("Age: %d\\n", age);
    printf("Height: %.2f meters\\n", height);

    return 0;
}

Sample Run:

Enter your age: 25
Enter your height: 1.75
Enter your name: Alice

Your Details:
Name: Alice
Age: 25
Height: 1.75 meters

Combining printf and scanf

In most programs, scanf and printf are used together to interact with users. Here’s an example:

Example:

#include <stdio.h>

int main() {
    int num1, num2, sum;

    printf("Enter two numbers: ");
    scanf("%d %d", &num1, &num2);

    sum = num1 + num2;

    printf("The sum of %d and %d is %d\\n", num1, num2, sum);

    return 0;
}

Sample Run:

Enter two numbers: 15 25
The sum of 15 and 25 is 40

Key Points

  • Always match the format specifier with the data type of the variable.
  • Use \\n to move to a new line in printf.
  • Do not forget the & operator in scanf for non-array variables.

Conclusion

The scanf and printf functions are powerful tools for input and output in C. They allow for dynamic interaction between the program and its users, making C programs versatile and functional.
