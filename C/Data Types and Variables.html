Data Types and Variables in C Language

Introduction

In C programming, data types define the type of data a variable can hold, while variables are used to store data values. Understanding these concepts is essential for efficient programming.

Data Types in C

C provides a variety of data types that are categorized into the following:

  • Primary Data Types: Basic types like int, float, char, and void.
  • Derived Data Types: Arrays, pointers, structures, and unions.
  • User-Defined Data Types: Created by the programmer using typedef or enum.
  • Enumeration: A set of named integer constants.

Primary Data Types

Data Type Size (in bytes) Description Example
int 4 Stores integer values int num = 10;
float 4 Stores floating-point numbers float pi = 3.14;
char 1 Stores single characters char grade = 'A';
void 0 Represents no value void function();

Variables in C

Variables are named storage locations in memory used to store data. Before using a variable, it must be declared with a specific data type.

Declaration and Initialization

A variable can be declared and initialized as follows:

int age;          // Declaration
age = 25;         // Initialization

// Or both can be done simultaneously:
int age = 25;

Rules for Naming Variables

  • Variable names must start with a letter or underscore.
  • They can contain letters, digits, and underscores.
  • Cannot use C keywords as variable names.
  • Case-sensitive (e.g., Var and var are different).

Scope and Lifetime of Variables

In C, variables have specific scopes and lifetimes:

  • Local Variables: Declared inside a function and accessible only within that function.
  • Global Variables: Declared outside all functions and accessible throughout the program.
  • Static Variables: Retain their value between function calls.
  • Extern Variables: Used to declare variables globally, accessible across multiple files.

Example Program

The following program demonstrates the use of variables and data types:

#include <stdio.h>

int main() {
    int num = 10;           // Integer variable
    float pi = 3.14;        // Float variable
    char grade = 'A';       // Character variable

    printf("Number: %d\n", num);
    printf("Value of Pi: %.2f\n", pi);
    printf("Grade: %c\n", grade);

    return 0;
}

Output:

  • Number: 10
  • Value of Pi: 3.14
  • Grade: A

Conclusion

Data types and variables are the building blocks of any C program. Understanding their usage, declaration, and scope is essential for writing effective and efficient programs.
