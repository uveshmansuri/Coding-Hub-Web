Structures in C

What is a Structure?

A structure in C is a user-defined data type that groups related variables of different types into a single unit. It is defined using the struct keyword and allows the programmer to create complex data types for efficient handling of real-world entities.

Features of Structures

  • Allows grouping of variables of different types.
  • Supports hierarchical data representation.
  • Provides a way to store records and implement complex data structures like linked lists.
  • Can include arrays, pointers, and even other structures as members.

Defining and Using Structures

The general syntax of defining a structure is:

struct StructureName {
    dataType member1;
    dataType member2;
    // Additional members
};

To declare variables of a structure:

struct StructureName variable1, variable2;

Example: Defining and Using a Structure

#include <stdio.h>

// Define a structure
struct Student {
    int id;
    char name[50];
    float marks;
};

int main() {
    struct Student student1;

    // Assign values to members
    student1.id = 101;
    strcpy(student1.name, "John Doe");
    student1.marks = 87.5;

    // Print details
    printf("Student Details:\n");
    printf("ID: %d\n", student1.id);
    printf("Name: %s\n", student1.name);
    printf("Marks: %.2f\n", student1.marks);

    return 0;
}

Nested Structures

Structures can be nested, meaning one structure can include another structure as a member. 

#include <stdio.h>

// Define a nested structure
struct Date {
    int day;
    int month;
    int year;
};

struct Student {
    int id;
    char name[50];
    struct Date dob; // Nested structure
};

int main() {
    struct Student student1;

    // Assign values
    student1.id = 101;
    strcpy(student1.name, "Jane Doe");
    student1.dob.day = 15;
    student1.dob.month = 6;
    student1.dob.year = 2000;

    // Print details
    printf("Student Details:\n");
    printf("ID: %d\n", student1.id);
    printf("Name: %s\n", student1.name);
    printf("Date of Birth: %02d/%02d/%d\n", student1.dob.day, student1.dob.month, student1.dob.year);

    return 0;
}

Array of Structures

Structures can be organized into arrays to store multiple records efficiently.

#include <stdio.h>

struct Student {
    int id;
    char name[50];
    float marks;
};

int main() {
    struct Student students[3];

    // Input data
    for (int i = 0; i < 3; i++) {
        printf("Enter details for student %d:\n", i + 1);
        printf("ID: ");
        scanf("%d", &students[i].id);
        printf("Name: ");
        scanf("%s", students[i].name);
        printf("Marks: ");
        scanf("%f", &students[i].marks);
    }

    // Display data
    printf("\nStudent Details:\n");
    for (int i = 0; i < 3; i++) {
        printf("ID: %d, Name: %s, Marks: %.2f\n", students[i].id, students[i].name, students[i].marks);
    }

    return 0;
}

Key Applications of Structures

  • Storing records like employee details, student information, or product data.
  • Building data structures like linked lists, stacks, and queues.
  • Facilitating interaction with hardware components in system-level programming.
