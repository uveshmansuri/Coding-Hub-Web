Looping Statements in C

Introduction

Looping is a programming concept that allows the execution of a set of instructions repeatedly until a certain condition is met. In C, loops are used to automate repetitive tasks. C provides three main types of looping statements: for loop, while loop, and do-while loop.

Types of Looping Statements in C

There are three primary looping statements in C:

  • For loop: It is used when the number of iterations is known beforehand.
  • While loop: It is used when the number of iterations is not known and the condition is checked before each iteration.
  • Do-while loop: It is similar to the while loop, but the condition is checked after each iteration, ensuring that the loop is executed at least once.

1. For Loop

The for loop is generally used when the number of iterations is known or can be determined easily. It includes an initialization, a condition, and an increment or decrement expression.

#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    return 0;
}

Output:

1 2 3 4 5

Explanation:

  • The loop starts with i = 1.
  • The loop runs while i <= 5.
  • After each iteration, i is incremented by 1.

2. While Loop

The while loop is used when the number of iterations is not known in advance. It checks the condition before executing the loop's body.

#include <stdio.h>

int main() {
    int i = 1;
    while (i <= 5) {
        printf("%d ", i);
        i++;
    }
    return 0;
}

Output:

1 2 3 4 5

Explanation:

  • The loop starts with i = 1.
  • The condition i <= 5 is checked before each iteration.
  • If the condition is true, the body of the loop is executed, and i is incremented.

3. Do-while Loop

The do-while loop is similar to the while loop, but it guarantees that the loop is executed at least once. The condition is checked after the loop's body.

#include <stdio.h>

int main() {
    int i = 1;
    do {
        printf("%d ", i);
        i++;
    } while (i <= 5);
    return 0;
}

Output:

1 2 3 4 5

Explanation:

  • The loop starts with i = 1.
  • The body of the loop is executed first, and then the condition i <= 5 is checked.
  • The loop continues to execute as long as the condition is true.

Control Statements: break, continue, and goto

In C, the break, continue, and goto statements allow you to control the flow of a loop in a more flexible manner.

break Statement

The break statement is used to exit a loop prematurely. It breaks out of the loop as soon as the condition is met, regardless of whether the loop's condition is still true.

#include <stdio.h>

int main() {
    for (int i = 1; i <= 10; i++) {
        if (i == 5) {
            break;  // Exits the loop when i equals 5
        }
        printf("%d ", i);
    }
    return 0;
}

Output:

1 2 3 4

Explanation:

  • When i == 5, the break statement is executed, and the loop terminates.

Continue Statement

The continue statement skips the current iteration and moves to the next iteration of the loop. It is often used to skip specific conditions inside the loop body.

#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            continue;  // Skips printing 3
        }
        printf("%d ", i);
    }
    return 0;
}

Output:

1 2 4 5

Explanation:

  • When i == 3, the continue statement is executed, skipping the printf statement for this iteration.

Goto Statement

The goto statement is used to transfer control to a specific label within the program. It is generally discouraged because it can make the code harder to understand and maintain.

#include <stdio.h>

int main() {
    int i = 1;
    start:
    if (i > 5) {
        return 0;
    }
    printf("%d ", i);
    i++;
    goto start;  // Jumps back to the 'start' label
}

Output:

1 2 3 4 5

Explanation:

  • The goto statement transfers control to the start label, causing the loop to repeat until i > 5.

Conclusion

Looping statements in C are powerful tools for automating repetitive tasks. Understanding when and how to use each type of loop allows developers to write efficient and effective programs. By combining loops with conditional statements, you can create complex, dynamic programs that perform a wide variety of tasks.
