Single-Dimensional Arrays in C

Introduction to Single-Dimensional Arrays

In C, an array is a collection of variables of the same type, stored in contiguous memory locations. A single-dimensional array (also known as a one-dimensional array) is a list of variables of the same type arranged in a linear sequence. The elements in an array are indexed starting from 0, which means the first element is accessed using index 0, the second element using index 1, and so on.

Arrays are useful for storing and manipulating collections of data, such as lists or sequences, and can be used in algorithms that require repetitive or similar tasks to be performed on multiple data elements.

Declaration and Initialization of Arrays

In C, arrays are declared by specifying the type of elements they will hold, followed by the array name, and the size of the array inside square brackets.

data_type array_name[array_size];

For example, to declare an array of 5 integers, you would write:

int numbers[5];

This will create an array of 5 integers, with the indices ranging from 0 to 4.

Initialization of Arrays

Arrays can be initialized at the time of declaration by providing values inside curly braces. For example:

int numbers[5] = {1, 2, 3, 4, 5};

If the number of initializers is less than the array size, the remaining elements will be initialized to zero:

int numbers[5] = {1, 2};  // numbers = {1, 2, 0, 0, 0}

If the size of the array is not specified during initialization, the compiler will automatically determine the size based on the number of elements provided:

int numbers[] = {1, 2, 3, 4, 5};  // Size is automatically 5

Accessing Array Elements

To access an element of an array, you use the array name followed by the index of the element inside square brackets. For example:

int numbers[5] = {1, 2, 3, 4, 5};
printf("%d", numbers[2]);  // Accessing the element at index 2 (output: 3)

Remember that array indices start at 0. So, numbers[0] will access the first element, numbers[1] the second, and so on.

Example of Using a Single-Dimensional Array

Here’s an example that demonstrates the declaration, initialization, and accessing of a single-dimensional array in C:

#include <stdio.h>

int main() {
    int numbers[5] = {10, 20, 30, 40, 50};
    
    // Accessing elements and printing them
    printf("First element: %d\n", numbers[0]);  // Output: 10
    printf("Second element: %d\n", numbers[1]); // Output: 20
    
    // Loop through the array and print all elements
    for (int i = 0; i < 5; i++) {
        printf("Element at index %d: %d\n", i, numbers[i]);
    }

    return 0;
}

Output:

First element: 10
Second element: 20
Element at index 0: 10
Element at index 1: 20
Element at index 2: 30
Element at index 3: 40
Element at index 4: 50

Array Indexing and Boundaries

C arrays are zero-indexed, meaning that the first element is accessed at index 0. If you attempt to access an array element with an invalid index (i.e., an index outside the bounds of the array), it can lead to undefined behavior, which might cause your program to crash or produce incorrect results.

For example, if you try to access numbers[5] in the above example, you will be accessing an out-of-bounds index, which is not defined in the array. Always ensure that your indices are within the valid range of the array (from 0 to array_size-1).

Example: Input and Output Using Arrays

Here’s an example where the user is prompted to enter 5 integers, which are then stored in an array and printed:

#include <stdio.h>

int main() {
    int numbers[5];
    
    // Taking input for the array
    printf("Enter 5 numbers:\n");
    for (int i = 0; i < 5; i++) {
        printf("Number %d: ", i + 1);
        scanf("%d", &numbers[i]);
    }
    
    // Displaying the array elements
    printf("\nYou entered the following numbers:\n");
    for (int i = 0; i < 5; i++) {
        printf("%d ", numbers[i]);
    }
    
    return 0;
}

Output:

Enter 5 numbers:
Number 1: 10
Number 2: 20
Number 3: 30
Number 4: 40
Number 5: 50

You entered the following numbers:
10 20 30 40 50

Array Passing to Functions

In C, arrays can be passed to functions either by value or by reference. However, due to the way arrays are implemented, when an array is passed to a function, it is always passed by reference. This means that any changes made to the array inside the function will affect the original array.

#include <stdio.h>

void modifyArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        arr[i] = arr[i] * 2;  // Double the value of each element
    }
}

int main() {
    int numbers[5] = {1, 2, 3, 4, 5};
    
    modifyArray(numbers, 5);  // Passing the array to the function
    
    printf("Modified array:\n");
    for (int i = 0; i < 5; i++) {
        printf("%d ", numbers[i]);
    }
    
    return 0;
}

Output:

Modified array:
2 4 6 8 10

Conclusion

Single-dimensional arrays are fundamental in C programming and provide an efficient way to store and manipulate data in a linear structure. They are widely used in algorithms, data processing, and system-level programming. Understanding how to properly declare, initialize, and access elements in an array is essential for every C programmer.
